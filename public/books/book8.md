## คำเตือนก่อนอ่าน

หนังสือเล่มนี้เป็นบันทึกจากประสบการณ์ส่วนตัว ไม่ใช่คำแนะนำการลงทุน ไม่ใช่สัญญาณซื้อขาย และไม่รับประกันผลลัพธ์ใด ๆ ทุก framework ตาราง และ checklist ในเล่มนี้ควรใช้เป็นเครื่องมือตั้งคำถามและสังเกต ไม่ใช่คำตอบสำเร็จรูป ผู้อ่านต้องประเมินความเสี่ยงด้วยตัวเอง และตัดสินใจตามแผน เงินทุน และความรับผิดชอบของตัวเอง

---

## เล่มนี้ไม่เหมือนเล่มอื่น

ถ้าคุณอ่านซีรีส์ "ผู้รอด" มาแล้ว — คุณรู้ว่าตลาดรู้สึกอย่างไร คุณรู้ว่าฤดูกาลเป็นอย่างไร คุณเห็นคนในตลาด เห็นตัวเอง เห็นวงจร

แต่คุณอาจยังไม่รู้ว่า — ต้อง "ทำ" อะไร

เล่มนี้ตอบคำถามนั้น

ไม่มีชาวนา ไม่มีท้องฟ้า ไม่มีอุปมา มีแต่ตาราง checklist diagram และคำสั่งตรง ๆ

ถ้าเล่ม ๑-๗ คือ "ทำไม" — เล่มนี้คือ "อย่างไร"

## 1.1 โทษตัวเองเท่านั้น

> ⚠️ WARNING: ถ้ายังโทษตลาด โทษอาจารย์ โทษนักวิเคราะห์ โทษข่าว — หยุดอ่านตรงนี้ กลับไปแก้จุดนี้ก่อน ไม่มี framework ไหนช่วยได้ถ้ายังโทษคนอื่น

คนที่เทรดแล้วขาดทุน มี 2 ประเภท:

**ประเภท A — โทษคนอื่น:**
- "อาจารย์บอกให้ซื้อ"
- "นักวิเคราะห์ปรับเป้าขึ้น"
- "กลุ่มไลน์ส่งมา"
- "ตลาดมันไม่ปกติ"
- "เจ้ามือเล่นอยู่"

**ประเภท B — โทษตัวเอง:**
- "ผมไม่ได้ตั้ง Stop Loss"
- "ผมใช้เงินเกินแผน"
- "ผมไม่ได้ดูงบก่อนซื้อ"
- "ผมไล่ราคา"
- "ผมไม่มีแผน"

ประเภท B เท่านั้นที่แก้ได้ เพราะ "ตัวเอง" เป็นสิ่งเดียวที่คุณควบคุมได้

```
┌─────────────────────────────────────────┐
│  KEY CONCEPT                            │
│                                         │
│  ฟ้าเริ่มเปิดเมื่อยอมรับว่า            │
│  เราเองที่ห่วย                          │
│                                         │
│  ไม่ใช่ตลาดห่วย                         │
│  ไม่ใช่อาจารย์ห่วย                      │
│  ไม่ใช่เครื่องมือห่วย                   │
│  เราห่วย — แล้วเราแก้ได้                │
└─────────────────────────────────────────┘
```

**ตัวอย่าง — จากประสบการณ์จริง:**

ปีที่สองในตลาด ผมขาดทุนสามเดือนติด บัญชีลดลงเกือบสามสิบเปอร์เซ็นต์ ผมโทษทุกอย่าง

"ตลาดมันไม่ปกติ" — ผมบอกตัวเอง

"อาจารย์คนนี้สอนไม่ดี" — ผมบอกเพื่อน

"ข่าวออกมาไม่ตรง" — ผมบอกคนที่บ้าน

จนวันหนึ่งผมนั่งเปิดดูทุกไม้ที่ขาดทุน ย้อนกลับไปดูว่า "ทำไมถึงซื้อ"

ไม้แรก — ซื้อเพราะกลุ่มไลน์ส่งมา ไม่ได้ดูกราฟเอง
ไม้สอง — ซื้อเพราะเห็นหุ้นวิ่ง กลัวตกรถ ไม่มี Stop Loss
ไม้สาม — ซื้อเพราะอาจารย์บอก ไม่ได้ดูงบเอง
ไม้สี่ — ซื้อเพราะ "รู้สึก" ว่าน่าจะขึ้น ไม่มีเหตุผล
ไม้ห้า — ซื้อเพราะอยากเอาคืนจากไม้สี่ เพิ่มขนาดเป็นสองเท่า

ทุกไม้ — ความผิดของผม ทุกไม้ ไม่มีข้อยกเว้น

วันนั้นผมหยุดโทษคนอื่น ไม่ใช่เพราะใจกว้าง เพราะ "ดูหลักฐานแล้วปฏิเสธไม่ได้"

หลังจากนั้น ทุกครั้งที่ขาดทุน ผมถามแค่คำถามเดียว: "ผมทำอะไรผิด" ไม่ถามว่า "ใครทำให้ผมผิด"

ฟ้าเริ่มเปิดตั้งแต่วันนั้น

---

**Checklist — ก่อนไปต่อ:**

☐ ผมยอมรับว่าทุกการขาดทุนที่ผ่านมาเป็นความรับผิดชอบของผม
☐ ผมหยุดตามสัญญาณจากคนอื่นโดยไม่คิดเอง
☐ ผมพร้อมที่จะเรียนรู้จากความผิดพลาดของตัวเอง

## 1.2 ชำระตะกอนทางจิตใจ — RESET เป็น 0

ก่อนจะเริ่มใหม่ ต้องล้างของเก่าก่อน

ตะกอน 4 ชนิดที่ต้องล้าง:

**ตะกอนที่ 1 — ความเสียดาย**

"ถ้าตอนนั้นไม่ขาย ตอนนี้ได้กำไร" — ประโยคนี้ไม่มีประโยชน์ สิ่งที่ผ่านไปแล้ว เปลี่ยนไม่ได้ ทุกวินาทีที่คิดเรื่องอดีตคือวินาทีที่ไม่ได้คิดเรื่องปัจจุบัน

**ตะกอนที่ 2 — ความกลัว**

"ครั้งก่อนขาดทุน ครั้งนี้จะขาดทุนอีก" — กลัวได้ แต่ถ้ากลัวจนไม่กล้าทำตามแผน กลัวจนตัดสินใจไม่ได้ กลัวจนไม่กดแม้เงื่อนไขครบ — ต้องล้าง

**ตะกอนที่ 3 — ความไม่มั่นใจ**

"ผมไม่รู้ว่าวิธีไหนถูก" — ปกติ ไม่มีใครรู้ 100% แต่ถ้าเปลี่ยนวิธีทุกอาทิตย์ ลองโน่นลองนี่ไม่จบ — ต้องหยุดแล้วเลือก

**ตะกอนที่ 4 — ความหลงตัวเอง**

"ผมรู้แล้ว ผมเก่งแล้ว" — ถ้าเก่งจริง พอร์ตจะบอก ไม่ต้องบอกตัวเอง

```
┌─────────────────────────────────────────┐
│  ACTION                                 │
│                                         │
│  วิธี RESET:                            │
│  1. ปิดพอร์ตเก่าในหัว — ไม่คิดถึงอีก   │
│  2. ปิดจอ 1 สัปดาห์ — ไม่ดูตลาดเลย     │
│  3. เขียนลงกระดาษ: "ผมเริ่มจาก 0"     │
│  4. กลับมาด้วยแผนใหม่ ไม่ใช่แผนเดิม    │
└─────────────────────────────────────────┘
```

**ตัวอย่าง — ก่อน RESET vs หลัง RESET:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: ก่อน RESET                                 │
├──────────────────────────────────────────────────────────┤
│  สถานการณ์: ขาดทุน 15% ใน 2 เดือน                       │
│  ความคิด: "ต้องเอาคืน" / "ไม่ยอมแพ้" / "เดี๋ยวมันกลับ" │
│  การกระทำ: เพิ่มขนาดเทรด เปลี่ยน product ไป Futures     │
│            ใช้ Margin เพิ่ม เทรดทุกวันไม่พัก              │
│  ผลลัพธ์: ขาดทุนเพิ่มเป็น 35% นอนไม่หลับ               │
│           ทะเลาะกับคนที่บ้าน                             │
└──────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: หลัง RESET                                 │
├──────────────────────────────────────────────────────────┤
│  สถานการณ์: ขาดทุน 8% ใน 1 เดือน                        │
│  ความคิด: "ผมทำอะไรผิด" / "ผมต้องหยุดก่อน"             │
│  การกระทำ: ปิดจอ 1 สัปดาห์ ทบทวนทุกไม้                  │
│            เขียนลงกระดาษ: สิ่งที่ทำถูก vs ทำผิด          │
│            กลับมาด้วยขนาดเล็กลงครึ่งหนึ่ง                │
│  ผลลัพธ์: เดือนถัดมากำไร 3% เดือนต่อมาอีก 4%           │
│           ไม่ได้เอาคืนทีเดียว แต่ค่อย ๆ กลับมา          │
└──────────────────────────────────────────────────────────┘
```

การ RESET ไม่ได้แปลว่า "ยอมแพ้" แปลว่า "หยุดทำสิ่งที่ไม่ได้ผล แล้วเริ่มใหม่ด้วยวิธีที่ดีกว่า"

## 1.3 เป้าหมายระยะสั้น — 5%/เดือน + Survivor Trade

> ⚠️ WARNING: อย่าตั้งเป้า "รวย" — ตั้งเป้า "รอด" ก่อน

**Survivor Trade คืออะไร:**

กำไรจากการเทรดครอบค่าใช้จ่ายรายเดือน โดยไม่แตะเงินต้น

```
ตัวอย่าง:
  เงินต้น          = 1,000,000 บาท
  ค่าใช้จ่าย/เดือน = 30,000 บาท
  เป้ากำไร/เดือน   = 50,000 บาท (5%)
  
  50,000 - 30,000 = 20,000 บาท → สะสมกลับเข้าพอร์ต
  เงินต้นไม่ลด เงินต้นค่อย ๆ โต
```

**ทำไมต้อง 5%:**
- ต่ำพอที่ไม่ต้องเสี่ยงเกิน
- สูงพอที่เลี้ยงชีพได้ (ถ้าพอร์ตใหญ่พอ)
- ทำได้จริงสม่ำเสมอ ≥ 6 เดือนติดกัน = เริ่มมั่นใจได้

**ถ้าพอร์ตยังเล็ก:**

ไม่ต้องลาออกจากงาน ทำ Survivor Trade เป็น "ฝึกซ้อม" — เงินเดือนจากงานเป็นค่าใช้จ่าย กำไรจากเทรดสะสมเข้าพอร์ต 100%

**ตัวอย่าง — Survivor Trade 3 ขนาดพอร์ต:**

```
┌──────────────────────────────────────────────────────────┐
│  พอร์ต 300,000 บาท — ยังทำงานประจำ                      │
├──────────────────────────────────────────────────────────┤
│  เป้า 5%/เดือน = 15,000 บาท                             │
│  ค่าใช้จ่าย = 0 (เงินเดือนจ่าย)                         │
│  สะสมเข้าพอร์ต = 15,000/เดือน                           │
│  12 เดือน = พอร์ตโต → ~480,000 (ถ้าทำได้สม่ำเสมอ)      │
│                                                          │
│  สถานะ: ฝึกซ้อม ไม่กดดัน ผิดพลาดได้                     │
└──────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│  พอร์ต 1,000,000 บาท — ยังทำงานประจำ                    │
├──────────────────────────────────────────────────────────┤
│  เป้า 5%/เดือน = 50,000 บาท                             │
│  ค่าใช้จ่าย = 0 (เงินเดือนจ่าย)                         │
│  สะสมเข้าพอร์ต = 50,000/เดือน                           │
│  12 เดือน = พอร์ตโต → ~1,600,000                        │
│                                                          │
│  สถานะ: เริ่มเห็นพลัง compound                           │
│  ยังไม่ต้องลาออก — รอจน 5% ครอบค่าใช้จ่ายได้ก่อน        │
└──────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│  พอร์ต 3,000,000 บาท — พิจารณาเทรดเต็มเวลา             │
├──────────────────────────────────────────────────────────┤
│  เป้า 5%/เดือน = 150,000 บาท                            │
│  ค่าใช้จ่าย/เดือน = 40,000 บาท                          │
│  เหลือสะสม = 110,000/เดือน                              │
│                                                          │
│  สถานะ: Survivor Trade ครอบค่าใช้จ่ายแล้ว               │
│  แต่! ต้องทำได้สม่ำเสมอ ≥ 6 เดือนก่อนลาออก             │
│  เพราะบางเดือนอาจได้ 2% บางเดือนอาจ -3%                 │
│  ต้องมีเงินสำรอง ≥ 6 เดือนค่าใช้จ่าย (240,000)         │
│  แยกไว้ต่างหาก ไม่ใช่ในพอร์ต                            │
└──────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────┐
│  KEY CONCEPT                            │
│                                         │
│  Survivor Trade ≠ รวย                   │
│  Survivor Trade = ไม่ตาย               │
│                                         │
│  รวยมาทีหลัง — ตอนที่ Compound         │
│  ทำงานให้แล้ว                           │
└─────────────────────────────────────────┘
```

## 1.5 Compound Interest — เส้นทาง 25%/ปี

ตัวเลขต่อไปนี้ไม่ใช่ความฝัน เป็นคณิตศาสตร์:

```
เงินต้น: 1,000,000 บาท | ผลตอบแทน: 25%/ปี | ทบต้น ไม่ถอน

  ปีที่  1  →   1,250,000
  ปีที่  2  →   1,562,500
  ปีที่  3  →   1,953,125
  ปีที่  5  →   3,051,758
  ปีที่  7  →   4,768,372
  ปีที่ 10  →   9,313,226
  ปีที่ 12  →  14,551,915
  ปีที่ 15  →  28,421,709
```

**เป้าหมายสุดท้าย:**

```
  28.42 ล้าน (nominal)
  ~17.9 ล้าน (หลังหักเงินเฟ้อ 3%)

  Passive Income 6-8%/ปี จาก 17.9 ล้าน
  = 1,074,000 - 1,432,000 บาท/ปี
  = 89,500 - 119,333 บาท/เดือน

  → Financial Freedom
```

```
┌─────────────────────────────────────────┐
│  KEY CONCEPT                            │
│                                         │
│  เงื่อนไขเดียว:                         │
│  คุณต้อง "อยู่ในตลาด" ให้ครบ 15 ปี     │
│                                         │
│  ไม่ใช่ทำ 25% ยาก                       │
│  อยู่ให้ครบ 15 ปี — นั่นแหละที่ยาก      │
└─────────────────────────────────────────┘
```

> ⚠️ WARNING: 25%/ปี ไม่ใช่ 25% ทุกปี — บางปีได้ 50% บางปีได้ -10% ค่าเฉลี่ยระยะยาวต้องอยู่ที่ ~25% ถ้าคุณเจ๊ง ปีที่เจ๊ง compound หยุดทำงาน ต้องเริ่มนับใหม่ — นี่คือเหตุผลที่ "ก่อนโต ต้องรอดก่อน"

**ตัวอย่าง — Compound vs เจ๊งแล้วเริ่มใหม่:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: คน A — Compound ไม่หยุด                    │
├──────────────────────────────────────────────────────────┤
│  เริ่ม: 1,000,000  เป้า: 25%/ปี  ไม่เจ๊งเลย            │

## 1.7 เครื่องมือ

เครื่องมือที่ใช้จริง — ไม่ต้องเยอะ ต้องเก่ง:

**Moving Average (EMA):**

| EMA | ใช้ดู |
|-----|------|
| EMA 5 | แนวโน้มสั้นมาก (1-3 วัน) |
| EMA 15 | แนวโน้มสั้น (1-2 สัปดาห์) |
| EMA 35 | แนวโน้มกลาง (1-2 เดือน) |
| EMA 89 | แนวโน้มยาว (3-6 เดือน) |

**กฎง่าย:**
- ราคาอยู่เหนือ EMA = ขาขึ้น
- ราคาอยู่ใต้ EMA = ขาลง
- EMA สั้นตัด EMA ยาวขึ้น = สัญญาณซื้อ
- EMA สั้นตัด EMA ยาวลง = สัญญาณขาย

**MACD:**
- ดู Divergence (ราคาทำ High ใหม่ แต่ MACD ไม่ตาม = อ่อนแรง)
- ดู Signal Line Cross (MACD ตัด Signal ขึ้น = momentum เปลี่ยน)

**RSI:**
- RSI > 70 = overbought (อาจพักตัว)
- RSI < 30 = oversold (อาจเด้ง)
- ดู Divergence เหมือน MACD

**Trendline:**
- ลากเส้นเชื่อม Low ที่สูงขึ้นเรื่อย ๆ = uptrend
- ลากเส้นเชื่อม High ที่ต่ำลงเรื่อย ๆ = downtrend
- เมื่อหลุด trendline = สัญญาณเปลี่ยนเทรนด์

**Fibonacci Retracement:**
- 38.2% / 50% / 61.8% = จุดพักตัวที่พบบ่อย
- ใช้หาจุดเข้า (ซื้อตอนพักตัวถึง Fibo level)

**Multiple Timeframe:**
- ดูภาพใหญ่ก่อน (Weekly/Daily) → หาจังหวะเข้าใน timeframe เล็ก (60min/15min)
- ทิศใหญ่บอกว่า "ไปทางไหน" — ทิศเล็กบอกว่า "เข้าเมื่อไหร่"

## 2.1 กล้าได้ กล้าเสีย

ตลาดมีอยู่ 2 ด้านเสมอ ได้กับเสีย คนที่เทรดได้ดีไม่ใช่คนที่ "ไม่เสีย" คือคนที่ "เสียแล้วไม่พัง"

ลองผิดลองถูกคือส่วนหนึ่งของกระบวนการ ถ้ากลัวเสียจนไม่กล้าซื้อตามแผน ไม่มีวันได้กำไร เพราะทุกไม้ที่ "ควรเข้า" แต่ไม่เข้า คือกำไรที่คุณทิ้งบนโต๊ะ

**กล้าได้:**
- เมื่อเงื่อนไขครบตามแผน — กด ไม่ลังเล
- ไม่กลัวที่จะ "ผิด" เพราะมี Stop Loss รองรับ
- ไม่ต้องรอจนแน่ใจ 100% — 60-70% ก็เข้าได้
- ถ้าแผนบอกว่าเข้า แต่มือไม่กด = แผนไม่มีความหมาย

**กล้าเสีย:**
- เมื่อถึง Stop Loss — กดขาย ไม่ต่อรอง
- ยอมรับขาดทุนเป็นส่วนหนึ่งของเกม
- ไม่ขาดทุนแล้วเอาคืน ไม่เพิ่มไม้ตอนผิดทาง
- Stop Loss ไม่ใช่ความพ่ายแพ้ — คือค่าใช้จ่ายในการเรียนรู้

```
┌─────────────────────────────────────────┐

## 2.2 โทษตัวเองเท่านั้น

เลิกโทษตลาด เลิกโทษอาจารย์ เลิกโทษนักวิเคราะห์ ฟ้าเริ่มเปิดเมื่อยอมรับว่าเราเองที่ห่วย

นี่ไม่ใช่การถ่อมตัว เป็นตรรกะ — เพราะ "ตัวเอง" เป็นสิ่งเดียวที่ควบคุมได้ ตลาดควบคุมไม่ได้ อาจารย์ควบคุมไม่ได้ แต่การตัดสินใจของตัวเองควบคุมได้

| สถานการณ์ | คนทั่วไปโทษ | ผู้รอดโทษ |
|-----------|------------|----------|
| ขาดทุนตามสัญญาณกลุ่มไลน์ | กลุ่มไลน์ | ตัวเองที่ไม่คิดเอง |
| ขาดทุนเพราะข่าวร้ายกะทันหัน | ข่าว | ตัวเองที่ไม่ตั้ง Stop Loss |
| ขาดทุนเพราะ "เจ้ามือ" | เจ้ามือ | ตัวเองที่เข้าหุ้นที่ไม่รู้จัก |
| ขาดทุนเพราะตลาดลงหมด | ตลาด | ตัวเองที่ไม่มี Circuit Breaker |
| ซื้อตามอาจารย์แล้วเสีย | อาจารย์ | ตัวเองที่ไม่ตรวจสอบก่อนซื้อ |

## 2.3 ควบคุมอารมณ์ให้นิ่ง

ตลาดทดสอบคุณ 2 ทิศ ทิศแรกคือดึงให้โลภ ทิศสองคือดึงให้กลัว คนที่ผ่านได้คือคนที่ไม่ไปกับทั้งสองทิศ

**อย่าหลงระเริงตอนได้:**
- กำไร 3 วันติด ≠ เก่ง อาจจะแค่ตลาดดี
- กำไรแล้วอย่าเพิ่มขนาดทันที ให้นิ่งก่อน
- กำไรแล้วอย่าเล่าให้ทุกคนฟัง มันทำให้ ego โต
- ช่วงที่ได้กำไรติดต่อกัน คือจุดอันตรายที่สุด เพราะความมั่นใจเกินจริงจะเข้ามา

**อย่าใจเสียตอนเสีย:**
- ขาดทุน 3 วันติด ≠ ห่วย อาจจะแค่ตลาดไม่ดี
- ขาดทุนแล้วอย่าเพิ่มไม้เอาคืน
- ขาดทุนแล้วอย่าปิดจอหนี — ดูให้รู้ว่า "ทำไม" แล้วจด
- ขาดทุนแล้วลดขนาดลง ไม่ใช่เพิ่มขนาด

```
┌───────────────────────────────────────────────────────┐
│  อารมณ์ตลาด — วงจรที่ต้องรู้                          │
│                                                       │
│  กำไร → ความมั่นใจเพิ่ม → เพิ่มขนาด                  │
│      → กำไรหรือขาดทุน?                               │
│                                                       │
│  ถ้ากำไร:                                             │
│  ความมั่นใจยิ่งสูง → เพิ่มขนาดอีก → ถึงจุดแตก        │
│                                                       │
│  ถ้าขาดทุน:                                           │
│  ตกใจ → เพิ่มไม้เอาคืน → ขาดทุนหนักขึ้น             │
│       → พัง                                           │
│                                                       │
│  ทางรอด:                                              │
│  กำไร → นิ่ง → ทำตามแผนเดิม                          │
│  ขาดทุน → นิ่ง → ลดขนาด → ทำตามแผนเดิม              │
└───────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────┐
│  ACTION                                 │
│                                         │
│  หลังเทรดทุกวัน ถามตัวเอง:             │
│                                         │
│  "วันนี้ผมตัดสินใจด้วยอารมณ์           │
│   หรือด้วยแผน?"                         │
│                                         │
│  ถ้าตอบว่า "อารมณ์" แม้แค่ครั้งเดียว    │
│  → พรุ่งนี้ลดขนาดลงครึ่ง               │
└─────────────────────────────────────────┘
```

**ตัวอย่าง — วันที่อารมณ์ชนะ vs วันที่แผนชนะ:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: วันจันทร์ — อารมณ์ชนะ                      │
├──────────────────────────────────────────────────────────┤
│  09:55 ตลาดเปิด: หุ้น A ที่ถือ ลง 4% ทันที              │
│  10:00 ตกใจ กดขาย ขาดทุน 4%                             │
│  10:30 หุ้น A เด้งกลับ +2%                               │
│  10:35 เสียดาย ซื้อกลับ ที่ราคาสูงกว่าเดิม               │
│  11:00 หุ้น A ลงอีก -3%                                  │
│  11:15 โกรธ ขายอีก ขาดทุนซ้ำ                             │
│  14:30 บ่ายเปิด: เห็นหุ้น B วิ่ง +8%                     │
│  14:35 กลัวตกรถ ซื้อหุ้น B ไล่ราคา                       │
│  16:00 หุ้น B พักตัว ลง -3% จากจุดที่ซื้อ                │
│                                                          │
│  สรุปวันนี้: ขาดทุน 3 ไม้ เหนื่อย ท้อ                   │
│  ทุกไม้ตัดสินใจจากอารมณ์ ไม่มีไม้ไหนตามแผน               │
└──────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: วันอังคาร — แผนชนะ                         │
├──────────────────────────────────────────────────────────┤
│  09:55 ตลาดเปิด: หุ้น A ที่ถือ ลง 4% ทันที              │
│  10:00 ดู Stop Loss → ยังไม่ถึง (ตั้งไว้ -5%)            │
│  10:00 ไม่ทำอะไร รอ                                      │
│  10:30 หุ้น A เด้งกลับ +2%                               │
│  10:30 ไม่ทำอะไร ตามแผน                                  │
│  14:30 หุ้น B วิ่ง +8%                                   │
│  14:30 ไม่อยู่ใน watchlist → ไม่เข้า                     │
│  16:30 ปิดตลาด: หุ้น A ปิด -1% → ยังไม่ถึง Stop         │
│                                                          │
│  สรุปวันนี้: ไม่เทรดเลย ไม่ได้กำไร แต่ไม่เสียเงิน      │
│  ทุกการตัดสินใจตามแผน                                    │
│  วันที่ดีที่สุด = วันที่ไม่ทำอะไรโง่                     │
└──────────────────────────────────────────────────────────┘
```

**สัญญาณเตือนว่าอารมณ์กำลังเข้ามา:**

| สัญญาณ | อารมณ์ที่ซ่อนอยู่ | ทำอะไร |
|--------|-----------------|--------|
| "ต้องซื้อตอนนี้เลย" | FOMO (กลัวตกรถ) | หยุด ดูแผนก่อน |
| "ถือต่อก่อน มันต้องขึ้น" | ความเสียดาย | ดู Stop Loss ทันที |
| "ครั้งนี้มั่นใจ 100%" | ความหลงตัวเอง | ลดขนาดลงครึ่ง |
| "เอาคืนให้ได้วันนี้" | ความโกรธ | ปิดจอ ออกไปเดิน |
| "ไม่ดูจอละ กลัวเกินไป" | ความกลัว | กลับมาดูแผน Stop Loss |

## 2.4 FOCUS

> คนจะรวยได้มาจากการ FOCUS เท่านั้น อย่าเป็นเป็ด

FOCUS ในที่นี้ไม่ใช่แค่ "ตั้งใจ" แต่คือ "เลือกสิ่งที่จะทำและไม่ทำอะไรนอกจากนั้น"

**ทำไม FOCUS ถึงสำคัญ:**

ทักษะการเทรดสร้างได้จาก pattern recognition — สมองต้องเห็น pattern เดิมซ้ำ ๆ หลายร้อยครั้งจึงจำได้ ถ้าคุณเปลี่ยน product ทุกเดือน สมองไม่ได้สะสม pattern ในอะไรเลย

```
┌───────────────────────────────────────────────────────┐
│  FOCUS Framework                                      │
│                                                       │
│  เลือก 1 Product:                                     │
│  หุ้น / DW / Futures — เลือก 1 ห้ามข้ามไปมา           │
│                                                       │
│  เลือก 1 Timeframe หลัก:                              │
│  Daily / 60min / 15min — ดูได้หลายอัน                  │
│  แต่ตัดสินใจจาก 1 อันเท่านั้น                         │
│                                                       │
│  เลือก 1 แบบการเทรด:                                  │
│  Daytrade / Swing / Position / ลงทุน                  │
│                                                       │
│  เลือก 3-5 หุ้นในพอร์ต watchlist:                     │
│  ศึกษาให้ลึก ไม่ใช่รู้แบน ๆ 50 ตัว                   │
└───────────────────────────────────────────────────────┘
```

| FOCUS | ไม่ FOCUS |
|-------|-----------|
| เลือก 1 เส้นทาง ฝึกจนนิ่ง | Daytrade วันจันทร์ Swing วันอังคาร |
| เลือก 2-3 เครื่องมือ | ใช้ 10 indicators พร้อมกัน |
| ดู 3-5 หุ้นที่ศึกษาจริง | เปิด watchlist 50 ตัว |
| เทรดเมื่อเงื่อนไขครบ | เทรดเพราะเบื่อ |
| ฝึก 6 เดือนก่อนเพิ่ม | เปลี่ยนวิธีทุกเดือน |

**กฎ FOCUS:**
1. เลือก 1 product (หุ้น / DW / Futures — อย่าข้ามไปมา)
2. เลือก 1 timeframe หลัก (ดูอื่นได้ แต่ตัดสินใจจาก 1 ตัว)
3. เลือก 1 แบบการเทรด (จาก 4 รูปแบบในภาค ๕)
4. ฝึก 6 เดือน ก่อนเพิ่ม

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: คนที่ FOCUS                                 │
├──────────────────────────────────────────────────────────┤
│  เดือนที่ 1-3:  เลือก Swing Trade + EMA 15/35           │
│                  ฝึก Paper Trade ดูแค่หุ้น SET50          │

│                  Hit Rate 55% → ปรับ → 62%               │

│                  Hit Rate 65% กำไรสม่ำเสมอ                │
│                                                          │
│  ผลลัพธ์: 1 ปี — ชำนาญ 1 วิธี ใช้หากินได้               │
└──────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: คนที่เป็นเป็ด                               │
├──────────────────────────────────────────────────────────┤
│  เดือนที่ 1:   ลอง Daytrade ด้วย RSI                    │
│  เดือนที่ 2:   เปลี่ยนไป Elliott Wave                    │

## 2.5 Discipline (วินัย)

วินัย ≠ ทำตามกฎเมื่อสบายใจ
วินัย = ทำตามกฎเมื่อไม่อยากทำ

ทุกคนมีวินัยในห้องที่มีคนดู ผู้รอดมีวินัยในห้องที่ไม่มีคนดู

ตลาดเป็นสถานที่ที่ไม่มีคนดู — ไม่มีเจ้านาย ไม่มีใครบังคับ มีแค่คุณกับปุ่มกด ในห้องนั้น วินัยของคุณคือคุณ

```
┌─────────────────────────────────────────┐
│  KEY CONCEPT                            │
│                                         │
│  วินัยไม่ได้ทำให้รวย                    │
│  วินัยทำให้ "ไม่ทำอะไรโง่"             │
│                                         │
│  และการไม่ทำอะไรโง่                     │
│  = กำไรที่ใหญ่ที่สุดในระยะยาว           │
└─────────────────────────────────────────┘
```

| สถานการณ์ | ไม่มีวินัย | มีวินัย |
|-----------|-----------|--------|
| หุ้นถึง Stop Loss แต่ "น่าจะเด้ง" | ถือต่อ | ขาย |
| หุ้นน่าซื้อแต่ยังไม่ถึงราคาเป้า | ซื้อเลย | รอ |
| กำไรแล้วแต่ยังไม่ถึงเป้า | ขายเอากำไร | ถือต่อตามแผน |
| วันนี้ไม่มีจังหวะ | หาจังหวะให้ได้ | ปิดจอ |
| พอร์ตลงเกิน Circuit Breaker | เทรดต่อเอาคืน | หยุด |

**ตัวอย่าง — วินัยในห้องที่ไม่มีคนดู:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: สองคน หนึ่งแผน                              │
├──────────────────────────────────────────────────────────┤
│  แผนเหมือนกัน: Swing Trade EMA 15/35                    │
│  Stop Loss: -5%  เป้ากำไร: +12%                          │
│                                                          │
│  คน A — มีวินัย:                                         │
│  สัปดาห์ที่ 1: ซื้อตามแผน → ถึง Stop → ขาย -5%         │
│  สัปดาห์ที่ 2: ซื้อตามแผน → ถึง Stop → ขาย -5%         │
│  สัปดาห์ที่ 3: ซื้อตามแผน → ขึ้นถึงเป้า → ขาย +12%    │
│  สัปดาห์ที่ 4: ซื้อตามแผน → ขึ้นถึงเป้า → ขาย +12%    │
│                                                          │
│  สรุป 4 สัปดาห์: -5 -5 +12 +12 = +14%                  │
│                                                          │
│  คน B — ไม่มีวินัย:                                      │
│  สัปดาห์ที่ 1: ซื้อตามแผน → ถึง Stop → "ถือต่อก่อน"     │
│               ราคาลงอีก → ขาย -12%                      │
│  สัปดาห์ที่ 2: เสียดาย → ไม่กล้าซื้อ พลาด +12%          │
│  สัปดาห์ที่ 3: กลับมาซื้อตามแผน → ถึงเป้า → ขาย +4%    │
│               (ขายก่อนเป้า เพราะกลัวจากสัปดาห์ที่ 1)    │
│  สัปดาห์ที่ 4: เห็นหุ้นอื่นวิ่ง ทิ้งแผนไปไล่ → -8%     │
│                                                          │
│  สรุป 4 สัปดาห์: -12 พลาด +4 -8 = -16%                 │
│                                                          │
│  แผนเดียวกัน: คน A +14% vs คน B -16%                   │
│  ต่างกันที่วินัย อย่างเดียว                               │
└──────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────┐
│  ACTION                                 │
│                                         │
│  เขียนแผนลงกระดาษก่อนตลาดเปิด          │
│  ระบุ: ซื้อที่เท่าไหร่ / Stop ที่เท่าไหร่│
│  / เป้าที่เท่าไหร่                       │
│                                         │
│  ระหว่างตลาดเปิด:                       │
│  ถ้าไม่อยู่ในแผน → ไม่เทรด              │
└─────────────────────────────────────────┘
```

## 2.6 อย่าเสียดายอดีต

> ผิดครั้งแรก = คนปกติ
> ผิดครั้งที่ 2 = คนใจดี
> ผิดครั้งที่ 3 = ต้องถามตัวเองแล้วว่ากำลังทำอะไรอยู่

อดีตเปลี่ยนไม่ได้ ทุกวินาทีที่คิดเรื่องอดีตคือวินาทีที่ไม่ได้คิดเรื่องปัจจุบัน ตลาดกำลังเคลื่อนไหวตอนนี้ — ไม่ใช่เมื่อวาน

ความเสียดายอดีตเป็นอาวุธที่อันตรายในตลาด เพราะมันทำให้คุณ:
- ถือของที่ไม่ควรถือ (รอให้กลับมาทุน)
- รอสิ่งที่ไม่มาถึง ("ถ้าขึ้นแค่นั้นจะขาย")
- เสียมากกว่าที่ควรเสีย (ขาดทุน 5% กลายเป็น 25%)

```
┌───────────────────────────────────────────────────────┐
│  วงจรความเสียดาย                                      │
│                                                       │
│  ขาดทุน 5% → "รอทุนก่อน"                             │
│           → ขาดทุน 10% → "รอ recovery"               │
│           → ขาดทุน 20% → "ตอนนี้รอไม่ได้แล้ว"        │
│           → ขาดทุน 30% → ขายด้วยความตกใจ            │
│                                                       │
│  ทางรอด:                                              │
│  ขาดทุน 5% → Stop Loss → ขาย → หาโอกาสถัดไป         │
└───────────────────────────────────────────────────────┘
```

**สิ่งที่ต้องล้างหน้าไพ่ใหม่ทันที:**
- หุ้นที่ถือแล้วเหตุผลเปลี่ยน — ขาย ไม่ต้องรอทุน
- วิธีที่ใช้แล้วขาดทุน 3 ครั้งติด — หยุดใช้ ทบทวน
- อาจารย์ที่ตามแล้วไม่เวิร์ค — หยุดตาม
- ความเชื่อเดิมที่พิสูจน์แล้วว่าผิด — ทิ้ง

**ตัวอย่าง — ต้นทุนของความเสียดาย:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: รอทุนจนพัง                                  │
├──────────────────────────────────────────────────────────┤
│  ซื้อหุ้น XYZ ที่ 40 บาท เหตุผล: "อาจารย์บอกว่าดี"      │
│                                                          │
│  ราคาลงมาที่ 36 บาท (-10%)                               │
│  คิด: "แค่ -10% รอทุนก่อน"                               │
│  ไม่มี Stop Loss                                         │
│                                                          │
│  ราคาลงมาที่ 32 บาท (-20%)                               │
│  คิด: "ลงมาเยอะแล้ว ต้องเด้งแน่"                         │
│  ยังถือ + เพิ่มไม้ที่ 32 เพื่อ "ลดต้นทุน"                │
│                                                          │
│  ราคาลงมาที่ 24 บาท (-40% จากที่ซื้อแรก)                │
│  ค่อยขาย เพราะ "ทนไม่ไหวแล้ว"                            │
│                                                          │
│  คำนวณ:                                                  │
│  ไม้แรก: ซื้อ 40 ขาย 24 = -40%                          │
│  ไม้สอง: ซื้อ 32 ขาย 24 = -25%                          │
│  ถ้าตั้ง Stop Loss ตั้งแต่ต้น (ที่ -8% = 36.80):        │
│  ขาดทุนแค่ -8% ไม้เดียว ไม่มีไม้สอง                     │
│                                                          │
│  ต้นทุนของความเสียดาย: -40% แทนที่จะเป็น -8%            │

## 2.7 อย่าใช้เงินเกินตัว

ตอนได้กำไรต่อเนื่องคือจุดอันตรายที่สุด เพราะความมั่นใจเกินจริงจะทำให้เพิ่มขนาดเกินแผน และเพิ่มขนาดในจังหวะที่ตลาดกำลังจะเปลี่ยนทิศ

**เงินที่ห้ามใช้เทรด:**
- เงินค่าเทอมลูก
- เงินผ่อนบ้าน / รถ
- เงินฉุกเฉิน
- เงินกู้ (ทุกประเภท)
- เงินที่ต้องใช้ภายใน 6 เดือน
- เงิน Margin (ยืมจากโบรกเกอร์) — ถ้ายังไม่ผ่านเฟส 5

**เงินที่ใช้เทรดได้:**
- เงินที่ถ้าหายไปทั้งก้อน ชีวิตไม่เปลี่ยน
- เงินที่ไม่มีใครมาถามว่า "อยู่ไหน"

```
┌─────────────────────────────────────────┐
│  ACTION                                 │
│                                         │
│  ก่อนเทรดทุกครั้ง ถาม:                  │
│  "ถ้าเงินก้อนนี้หายไป                   │
│   ครอบครัวจะเดือดร้อนไหม"               │
│                                         │
│  ถ้า "ไม่" → เข้าได้                    │
│  ถ้า "อาจจะ" → ลดขนาด                  │
│  ถ้า "ใช่" → ไม่เข้า                    │
└─────────────────────────────────────────┘
```

**กฎเงินเพิ่มเมื่อกำไรติดต่อกัน:**

```
┌───────────────────────────────────────────────────────┐
│  กำไร 3 เดือนติด → ตรวจสอบก่อนเพิ่มขนาด              │
│                                                       │
│  ถามตัวเอง 3 ข้อ:                                     │
│  1. ตลาดอยู่ในขาขึ้นใช่ไหม? (อาจเป็นเพราะตลาด)       │
│  2. Hit Rate จริง ๆ เป็นเท่าไหร่ใน 20 ไม้ล่าสุด?      │
│  3. กำไรมาจากแผน หรือมาจากโชค?                       │
│                                                       │
│  ตอบ 3/3 ได้แน่ใจ → เพิ่มขนาดได้ แต่ไม่เกิน +25%    │
│  ตอบไม่ได้ → รักษาขนาดเดิม                           │
└───────────────────────────────────────────────────────┘
```

**ตัวอย่าง — เงินเกินตัว:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: เทรดเดอร์ที่ใช้เงินเกินตัว                  │
├──────────────────────────────────────────────────────────┤
│  สถานการณ์:                                              │
│  - เงินเก็บ 500,000 บาท                                 │
│  - ค่าผ่อนบ้าน 15,000/เดือน                              │
│  - ค่าเทอมลูก 30,000/เทอม (อีก 2 เดือน)                 │
│  - ตัดสินใจใส่ 400,000 เข้าพอร์ต                         │
│                                                          │
│  สิ่งที่เกิดขึ้น:                                        │
│  - เดือน 1: ได้ 8% (+32,000) รู้สึกดี                    │
│  - เดือน 2: เสีย 12% (-51,840) ตกใจ                     │
│  - ค่าเทอมลูกถึงกำหนด เงินไม่พอ                         │

│  - ขาดทุนซ้ำ เพราะขายตอนที่ไม่ควรขาย                    │
│                                                          │
│  ปัญหาจริง:                                              │
│  ไม่ใช่ "เทรดไม่เก่ง" — คือ "ใช้เงินที่ห้ามเสีย"        │
│  ถ้าใช้แค่ 200,000 (เงินที่เสียได้)                      │
│  → ไม่ต้อง force sell → ไม่ขาดทุนซ้ำ                    │
│  → ค่าเทอมลูกจ่ายได้ → ไม่เครียด → ตัดสินใจดีขึ้น       │
└──────────────────────────────────────────────────────────┘
```

```

│                                                          │

## 2.8 อย่าคิดว่าตัวเองแน่

> ถ้าเก่งจริงพอร์ตจะบอก ไม่ต้องบอกตัวเอง

ความคิดว่าตัวเองแน่เป็น "เชื้อเพลิง" ให้กับความเสียหายครั้งใหญ่ เพราะทำให้:
- ลด Guard ลง
- เพิ่มขนาดเกินแผน
- ข้ามขั้นตอน
- ไม่ตั้ง Stop Loss

**สัญญาณว่าคุณเริ่ม "คิดว่าแน่":**
- เพิ่มขนาดเทรดเพราะ "มั่นใจ" ไม่ใช่เพราะ "แผน"
- ไม่ตั้ง Stop Loss เพราะ "ไม่มีทางผิด"
- เทรดหุ้นที่ไม่ได้ศึกษาเพราะ "สัญชาตญาณบอก"
- เริ่มสอนคนอื่น ทั้งที่ยังไม่ผ่าน 1 ปีเต็ม
- กำไรแล้วเชื่อว่า "เพราะเก่ง" ไม่ใช่ "เพราะตลาดดี"

```
┌─────────────────────────────────────────┐
│  KEY CONCEPT                            │
│                                         │
│  ฤดูร้อน ทุกคนเป็นอัจฉริยะ             │
│  เพราะน้ำขึ้น ทุกสำเภาลอย              │
│                                         │
│  เมื่อน้ำลง — จะเห็นว่าใครว่ายน้ำเป็น  │
│  ใครลอยเพราะน้ำอุ้ม                     │
└─────────────────────────────────────────┘
```

**ตัวอย่าง — ฤดูร้อนที่หลอกตา:**

```

## 2.9 อย่าเปรียบเทียบกับคนอื่น

แต่ละคนเริ่มต่างกัน เงินต้นต่างกัน ระบบต่างกัน ระดับความเสี่ยงต่างกัน เปรียบเทียบตัวเองกับ "ตัวเองเมื่อวาน" เท่านั้น

| เปรียบเทียบ | ผลลัพธ์ |
|-------------|---------|
| "เพื่อนได้ 20% เดือนนี้" | คุณเพิ่มความเสี่ยงไล่ตาม |
| "คนนั้นเทรด DW ได้เยอะ" | คุณกระโดดไป DW โดยไม่รู้จัก |
| "กลุ่มไลน์คนนี้ได้ทุกวัน" | คุณเชื่อคนอื่นแทนเชื่อแผนตัวเอง |
| "อาจารย์บอกว่ามีคนได้ 100%" | คุณฝันว่าน่าจะทำได้บ้าง |

**สิ่งเดียวที่ควรเปรียบเทียบ:**
- ตัวเอง วันนี้ vs ตัวเอง เมื่อ 6 เดือนก่อน
- ถ้าดีขึ้น — ไปต่อ
- ถ้าเหมือนเดิม — ทบทวน
- ถ้าแย่ลง — หยุดแล้วแก้

**ตัวอย่าง — โซเชียลมีเดียกับการเปรียบเทียบ:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: กับดักโซเชียลมีเดีย                         │
├──────────────────────────────────────────────────────────┤
│  สิ่งที่คุณเห็น:                                         │
│  - คนในกลุ่มโพสต์กำไร 30% ทุกเดือน                      │
│  - อาจารย์โชว์พอร์ต +150% ใน 6 เดือน                    │
│  - เพื่อนเล่า "เข้าตรงนั้น ได้เท่านี้ง่ายมาก"            │
│                                                          │
│  สิ่งที่คุณไม่เห็น:                                       │
│  - คนที่ขาดทุนไม่โพสต์ มีแต่คนกำไรที่โพสต์              │

│  ถ้าเปรียบตัวเองกับ "คนที่โพสต์"                         │

# ภาค ๓ — Stock Quadrant (SQ)

## 3.1 รหัสหน้า — พื้นฐาน (A / B / C)

ก่อนซื้อหุ้นตัวไหน ต้องให้ "รหัส" มันก่อน รหัสมี 2 ตัว: ตัวหน้า = พื้นฐาน ตัวหลัง = เทรนด์

ไม่มีหุ้นไหนที่ดีหรือแย่ตลอดกาล รหัสเปลี่ยนได้เมื่อข้อมูลเปลี่ยน งานของคุณคืออัพเดตรหัสให้ตรงกับความเป็นจริงปัจจุบัน ไม่ใช่ตามอดีต

**รหัสหน้า — ประเมินพื้นฐาน:**

| รหัส | หมายความ | เกณฑ์ |
|------|---------|-------|
| **A** | พื้นฐานดี | กำไรเติบโต ≥ 3 ไตรมาสติด / หนี้ไม่เพิ่ม / P/E ไม่สูงเกินกลุ่ม / ผู้บริหารยังถือ / กองทุนยังอยู่ |
| **B** | พื้นฐานปานกลาง | กำไรมี แต่ไม่โดดเด่น / อยู่ในกลุ่มที่ไม่ร้อนไม่เย็น / ไม่มีปัจจัยบวก-ลบชัดเจน |
| **C** | พื้นฐานแย่ | กำไรลด / ขาดทุน / หนี้เพิ่ม / ผู้บริหารขาย / ธุรกิจมีปัญหา |

**วิธีประเมินเร็ว 5 นาที — Quick Screen:**

| ข้อ | ดู | ดี (บวก) | ระวัง (ลบ) |
|-----|-----|---------|----------|
| 1 | กำไรสุทธิ 4Q ล่าสุด | เติบโตทุก Q | ลด หรือสลับกำไร-ขาดทุน |
| 2 | P/E | ต่ำกว่าค่าเฉลี่ยกลุ่ม | สูงกว่ากลุ่ม > 50% |
| 3 | P/BV | < 1.5 หรือต่ำกว่า 1 = ถูก | > 3 = แพง |
| 4 | D/E (หนี้สิน/ทุน) | < 1 | > 2 |
| 5 | ปันผล / ผู้ถือหุ้นรายใหญ่ | จ่ายสม่ำเสมอ / รายใหญ่เพิ่ม | ไม่จ่าย / รายใหญ่ลด |

ตอบ 4/5 ข้อเป็นบวก = A
ตอบ 2-3 ข้อเป็นบวก = B
ตอบ 0-1 ข้อเป็นบวก = C

## 3.2 รหัสหลัง — เทรนด์ (A / B / C)

รหัสหลังบอกว่า "ตอนนี้ตลาดคิดอะไรกับหุ้นตัวนี้" พื้นฐานดีแต่ตลาดยังไม่สนใจ = AB ไม่ใช่เวลาเข้า

| รหัส | หมายความ | เกณฑ์ |
|------|---------|-------|
| **A** | ขาขึ้น | ราคาอยู่เหนือ EMA 35 / Higher High + Higher Low / Volume เพิ่มเมื่อขึ้น / EMA 15 อยู่เหนือ EMA 35 |
| **B** | ไซด์เวย์ | ราคาแกว่งในกรอบ / ไม่มี Higher High หรือ Lower Low ชัด / Volume ต่ำ / EMA 15 และ 35 พันกัน |
| **C** | ขาลง | ราคาอยู่ใต้ EMA 35 / Lower High + Lower Low / Volume เพิ่มเมื่อลง / EMA 15 อยู่ใต้ EMA 35 |

**วิธีอ่านเทรนด์เร็ว 3 นาที:**

```
┌───────────────────────────────────────────────────────┐
│  เปิดกราฟ Daily + ใส่ EMA 15 และ EMA 35              │
│                                                       │
│  ขั้นที่ 1: EMA 15 อยู่เหนือ EMA 35 ไหม?             │
│  ใช่ = ขาขึ้น (เทรนด์ A ชั่วคราว)                    │
│  ไม่ใช่ = ขาลง หรือไซด์เวย์                          │
│                                                       │
│  ขั้นที่ 2: ราคาอยู่เหนือ EMA 35 ไหม?                │
│  ใช่ = ยืนยัน A                                      │
│  ไม่ใช่ = B หรือ C                                   │
│                                                       │
│  ขั้นที่ 3: High-Low ล่าสุด 3 จุด                    │
│  H1 < H2 < H3 และ L1 < L2 < L3 = A (ขาขึ้น)         │
│  H1 ≈ H2 ≈ H3 และ L1 ≈ L2 ≈ L3 = B (ไซด์เวย์)       │
│  H1 > H2 > H3 และ L1 > L2 > L3 = C (ขาลง)           │
└───────────────────────────────────────────────────────┘
```

## 3.3 ตาราง 9 ช่อง — SQ Matrix

เมื่อมีรหัสหน้า + รหัสหลัง → รวมกันเป็น SQ (Stock Quadrant) ตาราง 9 ช่อง

```
┌─────────────────────────────────────────────────────────────┐
│                    STOCK QUADRANT MATRIX                    │
├──────────┬──────────────┬──────────────┬────────────────────┤
│          │  เทรนด์ A    │  เทรนด์ B    │  เทรนด์ C          │
│          │  (ขาขึ้น)    │  (ไซด์เวย์)  │  (ขาลง)            │
├──────────┼──────────────┼──────────────┼────────────────────┤
│ พื้นฐาน  │     AA       │     AB       │     AC             │
│ A (ดี)   │  ALL-IN      │  แนวรับ-ต้าน │  ถ้า P/E ต่ำมาก    │
│          │  เต็มหน้าตัก  │  Channel     │  P/BV < 1 → DCA   │
├──────────┼──────────────┼──────────────┼────────────────────┤
│ พื้นฐาน  │     BA       │     BB       │     BC             │
│ B (กลาง) │  เข้า 50%   │  Channel     │  หลีกเลี่ยง        │
│          │  Cut Loss     │  Trade       │  ถ้าจะเข้า ต้องมี  │
│          │  เร็ว         │  ซื้อล่าง    │  catalyst ชัดมาก   │
│          │              │  ขายบน       │                    │
├──────────┼──────────────┼──────────────┼────────────────────┤
│ พื้นฐาน  │     CA       │     CB       │     CC             │
│ C (แย่)  │  เข้าไว      │  หลีกเลี่ยง  │  หลีกเลี่ยง        │
│          │  ออกไว        │              │  อย่างแน่วแน่       │
│          │  แค่ 10%     │              │                    │
└──────────┴──────────────┴──────────────┴────────────────────┘
```

**คำอธิบายแต่ละช่อง:**

**AA — พื้นฐานดี + ขาขึ้น**
- โอกาสที่ดีที่สุด ทำตามแผนทุกอย่าง
- เข้าเต็มหน้าตัก (30-50% ของพอร์ต)
- Stop Loss กว้างได้ (8-10%) เพราะพื้นฐานรองรับ
- ถือได้นานถ้าเทรนด์ยังดี

**AB — พื้นฐานดี + ไซด์เวย์**
- บริษัทดี แต่ตลาดยังไม่สนใจ
- ซื้อขายตามแนวรับ-แนวต้าน รอจังหวะ Break
- เข้า 15-25% รอ Break ขึ้น → ขยาย position

**AC — พื้นฐานดี + ขาลง**
- หุ้นคุณค่าที่ตลาดกำลังขาย
- ถ้า P/E ต่ำมาก P/BV < 1 → ทยอยซื้อ DCA
- เข้า 10-15% ทยอยไม่ซื้อทีเดียว

**BA — พื้นฐานกลาง + ขาขึ้น**
- กราฟดี แต่พื้นฐานปานกลาง
- เข้า 50% ของ full size (15-20% ของพอร์ต)
- Cut Loss เร็วถ้า Break ขึ้นผิดพลาด

**BB — พื้นฐานกลาง + ไซด์เวย์**
- Channel Trade ซื้อแนวรับ ขายแนวต้าน
- เข้า 10-15% เล่นสั้น

**BC — พื้นฐานกลาง + ขาลง**
- หลีกเลี่ยง ถ้าจะเข้าต้องมี catalyst ชัดมากจริง ๆ

**CA — พื้นฐานแย่ + ขาขึ้น**
- หุ้นเก็งกำไร momentum สั้น
- เข้าไว ออกไว ใช้เงินแค่ 5-10%
- Stop Loss แคบมาก (3%) ไม่ถือข้ามคืนนาน

**CB — พื้นฐานแย่ + ไซด์เวย์**
- หลีกเลี่ยง ไม่มีเหตุผลให้เข้า

**CC — พื้นฐานแย่ + ขาลง**
- หลีกเลี่ยงอย่างแน่วแน่ — ไม่มีข้อยกเว้น
- "หุ้นถูก" ไม่ได้แปลว่าน่าซื้อ ถ้า CC อาจถูกลงอีก

```
┌─────────────────────────────────────────┐
│  KEY CONCEPT                            │
│                                         │
│  SQ คือระบบกรองก่อนเทรด               │
│                                         │
│  AA = สีเขียวสว่าง → เข้าเต็มที่        │
│  AB/BA = สีเขียวอ่อน → เข้าครึ่ง       │
│  BB/AC = สีเหลือง → เข้าน้อย/รอ       │
│  BC/CA = สีส้ม → ระวัง                 │
│  CB/CC = สีแดง → ไม่เข้า              │
└─────────────────────────────────────────┘
```

**ตัวอย่าง — จัด SQ ก่อนตัดสินใจ:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: Watchlist 5 ตัว → เลือก 2 ตัว              │
├──────────────────────────────────────────────────────────┤
│  สถานการณ์: มีเงิน 500,000 บาท พร้อมเทรด               │
│                                                          │
│  จัด SQ:                                                 │
│  หุ้น A: พื้นฐาน A (กำไรโต 4Q) + เทรนด์ A = AA         │

## 3.4 Money Management ตาม SQ

**หลักคิด:** ยิ่งรหัสดี ยิ่งใส่เงินได้เยอะ ยิ่งรหัสแย่ ยิ่งใส่น้อยหรือไม่ใส่เลย

| SQ | % ของพอร์ตที่ใส่ได้ | Stop Loss | หมายเหตุ |
|----|---------------------|-----------|----------|
| **AA** | 30-50% | กว้างได้ (8-10%) | หุ้นดี + เทรนด์ดี = โอกาสดีที่สุด |
| **AB** | 15-25% | ปานกลาง (5-7%) | ซื้อแนวรับ ขายแนวต้าน |
| **AC** | 10-15% | กว้าง (10-15%) | DCA ทยอยซื้อ ไม่ซื้อทีเดียว |
| **BA** | 15-20% | แคบ (5%) | กระจายเงิน อย่าใส่มาก |
| **BB** | 10-15% | แคบ (3-5%) | Channel trade ซื้อล่าง ขายบน |
| **BC** | 0% | - | ไม่เข้า |
| **CA** | 5-10% | แคบมาก (3%) | เข้าไว ออกไว ห้ามถือนาน |
| **CB** | 0% | - | ไม่เข้า |
| **CC** | 0% | - | ไม่เข้า |

**Sniper Shot — กฎ ≤ 3 ตัว:**

```
┌─────────────────────────────────────────┐
│  KEY CONCEPT                            │
│                                         │
│  Sniper Shot: เทรดไม่เกิน 3 ตัว        │
│  เลือกขาขึ้นเท่านั้น (เทรนด์ A)        │

**ตัวอย่าง — คำนวณ Position Size ตาม SQ:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: พอร์ต 1,000,000 บาท / SQ ต่างกัน 3 ตัว    │
├──────────────────────────────────────────────────────────┤
│  หุ้น X (SQ = AA):                                       │
│  Position Size = 35% = 350,000 บาท                      │
│  Stop Loss = -8% → ถ้า SL hit ขาดทุน = 28,000 บาท       │
│  เป้ากำไร = +15% = 52,500 บาท                           │
│  RRR = 52,500 / 28,000 = 1.875 ✓ (ควรได้ ≥ 2)           │
│                                                          │
│  หุ้น Y (SQ = BA):                                       │
│  Position Size = 15% = 150,000 บาท                      │
│  Stop Loss = -5% → ถ้า SL hit ขาดทุน = 7,500 บาท        │
│  เป้ากำไร = +10% = 15,000 บาท                           │
│  RRR = 15,000 / 7,500 = 2.0 ✓                           │
│                                                          │
│  หุ้น Z (SQ = CA):                                       │
│  Position Size = 8% = 80,000 บาท                        │
│  Stop Loss = -3% → ถ้า SL hit ขาดทุน = 2,400 บาท        │
│  เป้ากำไร = +8% = 6,400 บาท                             │
│  RRR = 6,400 / 2,400 = 2.67 ✓                           │
│                                                          │
│  รวมใช้ทุน: 350+150+80 = 580,000 บาท (58% ของพอร์ต)    │
│  เงินสด: 420,000 บาท (42%)                              │
│  ถ้า SL ทุกตัว hit พร้อมกัน: ขาดทุนสูงสุด 37,900 บาท   │
│  = 3.79% ของพอร์ต — รับได้                              │
└──────────────────────────────────────────────────────────┘
```

**กรณีพิเศษ — SQ เปลี่ยนระหว่างถือ:**

```
┌───────────────────────────────────────────────────────┐
│  เมื่อ SQ เปลี่ยน → ขนาดต้องเปลี่ยนตาม               │
│                                                       │
│  AA → AB (เทรนด์พักตัว):                             │
│  ลดขนาดจาก 35% → 20% ขายบางส่วนออกทันที             │
│                                                       │
│  AA → AC (เทรนด์กลายเป็นลง):                         │
│  ลดขนาดจาก 35% → 10% หรือออกหมดถ้า P/BV ไม่ต่ำ    │
│                                                       │
│  AB → AA (ไซด์เวย์แล้ว Break ขึ้น):                  │
│  เพิ่มขนาดจาก 20% → 35% ได้เลย                      │
└───────────────────────────────────────────────────────┘
```

## 3.5 ตัวอย่างจริง — วิธีจัด SQ ก่อนเทรด

**ขั้นตอนครบวงจร:**

```
1. เปิด watchlist → เลือกหุ้น 5-10 ตัวที่สนใจ
2. ให้รหัสหน้า (พื้นฐาน) แต่ละตัว — Quick Screen 5 ข้อ
3. ให้รหัสหลัง (เทรนด์) แต่ละตัว — ดูกราฟ 3 นาที
4. จัดเข้าตาราง SQ Matrix
5. เลือกเฉพาะ AA / AB / BA — ไม่เกิน 3 ตัว
6. กำหนด % ของพอร์ตตาม SQ
7. กำหนด Stop Loss ตาม SQ
8. กำหนดเป้ากำไร (ต้อง RRR ≥ 2)
9. เข้าเทรด
```

**ตัวอย่างครบวงจร:**

| หุ้น | รหัสหน้า | รหัสหลัง | SQ | action |
|------|---------|---------|-----|--------|
| หุ้น A | A (กำไรโต 4Q) | A (เหนือ EMA 35) | AA | เข้า 30% / SL -8% |
| หุ้น B | B (กำไรปกติ) | A (เพิ่ง break) | BA | เข้า 15% / SL -5% |
| หุ้น C | A (กำไรดี) | C (ใต้ EMA 35) | AC | ยังไม่เข้า — รอ DCA ถ้า P/BV < 1 |
| หุ้น D | C (ขาดทุน) | A (ขึ้นแรง) | CA | ไม่เข้า (พื้นฐานแย่เกินไป) |
| หุ้น E | C (หนี้เพิ่ม) | C (ขาลง) | CC | ไม่เข้า |

**สรุป: เข้า 2 ตัว** (หุ้น A + หุ้น B) ใช้ 45% ของพอร์ต เหลือเงินสด 55%

**ตัวอย่าง — SQ เปลี่ยนระหว่างถือ:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: หุ้นที่เปลี่ยน SQ จาก AA → AC              │
├──────────────────────────────────────────────────────────┤
│  สัปดาห์ที่ 1:                                           │
│  รหัส SQ = AA (กำไรโต + เทรนด์ขาขึ้น)                   │
│  ซื้อ 30% ของพอร์ต ที่ราคา 25 บาท                       │
│  Stop Loss ที่ 23 บาท (-8%)                              │
│                                                          │
│  สัปดาห์ที่ 4:                                           │
│  ราคาขึ้นไป 30 บาท (+20%) กำไรดี                         │
│  SQ ยังเป็น AA → ถือต่อ ขยับ Stop Loss ขึ้นมาที่ 27      │
│                                                          │
│  สัปดาห์ที่ 8:                                           │
│  ราคาหลุด EMA 35 → เทรนด์เปลี่ยนจาก A เป็น C           │
│  SQ เปลี่ยนจาก AA → AC                                  │
│                                                          │
│  ถาม: ทำอะไร?                                            │
│                                                          │
│  ผิด: "ก็ยังกำไรอยู่ ถือต่อก่อน"                         │
│  ถูก: SQ เปลี่ยนแล้ว → ลดขนาดลง 50% ทันที              │
│       → ขยับ Stop Loss แน่นขึ้น (28 บาท)                │
│       → ถ้า AC + P/BV ยังต่ำ → ถือส่วนที่เหลือเป็น DCA  │
│       → ถ้า AC + P/BV สูง → ออกหมด                      │
│                                                          │
│  หลักการ: SQ เป็นเข็มทิศ เมื่อเข็มเปลี่ยนทิศ           │
│  คุณต้องเปลี่ยนตาม ไม่ใช่ยืนยันว่าเข็มผิด               │
└──────────────────────────────────────────────────────────┘
```

**5 ข้อผิดพลาดที่พบบ่อยที่สุดกับ SQ:**

```
┌──────────────────────────────────────────────────────────┐
│  5 ผิดพลาดที่พบบ่อย                                     │
├────┬─────────────────────────────────────────────────────┤
│ 1  │ ให้รหัส A เพราะ "อยากซื้อ" ไม่ใช่เพราะข้อมูล      │
│    │ → ถ้าต้อง "หาเหตุผล" ให้มันเป็น A = ไม่ใช่ A      │
├────┼─────────────────────────────────────────────────────┤
│ 2  │ ไม่อัพเดต SQ เมื่องบออกใหม่                        │
│    │ → SQ ที่ไม่อัพเดต = แผนที่เก่า นำทางผิด             │
├────┼─────────────────────────────────────────────────────┤
│ 3  │ เข้า BC/CB/CC เพราะ "หุ้นน่าจะเด้ง"                │
│    │ → SQ บอกหลีกเลี่ยง = หลีกเลี่ยง ไม่มีข้อยกเว้น    │
├────┼─────────────────────────────────────────────────────┤
│ 4  │ ถือ 5 ตัว เพราะ "ทุกตัวเป็น AA"                    │
│    │ → Sniper Shot = ≤ 3 ตัว แม้ AA ทุกตัว              │
│    │ → เลือก 3 ตัวที่ดีที่สุดจาก 5                       │
├────┼─────────────────────────────────────────────────────┤
│ 5  │ ใส่เงินเท่ากันทุก SQ                                │
│    │ → AA ได้ 30-50% แต่ BA ได้แค่ 15-20%               │
│    │ → SQ ต่างกัน ขนาดต้องต่างกัน                        │
└────┴─────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────┐
│  ACTION                                 │
│                                         │
│  ทำ SQ ทุกสัปดาห์                       │
│  อัพเดตรหัสเมื่อมีข้อมูลใหม่            │
│  (งบใหม่ → เปลี่ยนรหัสหน้า             │
│   เทรนด์เปลี่ยน → เปลี่ยนรหัสหลัง)     │
│                                         │

└─────────────────────────────────────────┘
```

## 4.1 ขั้นตอนที่ 1: มองสถานการณ์ → ธุรกิจอะไรได้ประโยชน์

ก่อนหาหุ้น หาสถานการณ์ก่อน อย่าเริ่มจากหุ้นที่ชอบ เริ่มจากสถานการณ์ที่เกิดขึ้น แล้วถามว่า "ธุรกิจไหนได้ประโยชน์จากสถานการณ์นี้"

การคิดแบบ Top-Down ไม่ใช่แค่ทำให้เจอหุ้นได้ แต่ทำให้เข้าหุ้นที่มีเหตุผลชัดเจนว่าทำไมมันจะขึ้น ซึ่งทำให้ถือได้มั่นใจกว่า

**สถานการณ์ → กลุ่มที่ได้ประโยชน์:**

| สถานการณ์ | กลุ่มที่ได้ | กลุ่มที่เสีย |
|-----------|-----------|-------------|
| บาทอ่อน | ส่งออก (อาหาร, อิเล็กทรอนิกส์, ยาง) | นำเข้า (น้ำมัน, วัตถุดิบ) |
| บาทแข็ง | นำเข้า, ท่องเที่ยว outbound | ส่งออก |
| ดอกเบี้ยลง | อสังหาฯ, ธนาคาร (สินเชื่อโต), REITs | ประกัน (yield ลด) |
| ดอกเบี้ยขึ้น | ธนาคาร (NIM โต), ประกัน | อสังหาฯ (สินเชื่อแพง) |
| น้ำมันขึ้น | พลังงาน, ปิโตรเคมี | สายการบิน, ขนส่ง |
| น้ำมันลง | สายการบิน, ขนส่ง, โรงไฟฟ้า | พลังงาน |
| นักท่องเที่ยวเข้าเยอะ | โรงแรม, ค้าปลีก, สนามบิน | - |
| ภัยธรรมชาติ | วัสดุก่อสร้าง (ฟื้นฟู) | ประกันภัย |
| รัฐลงทุนโครงสร้างพื้นฐาน | รับเหมาก่อสร้าง, วัสดุก่อสร้าง | - |
| เงินเฟ้อสูง | สินค้าโภคภัณฑ์, ทองคำ, อสังหาฯ | หุ้นเติบโต (discount rate สูง) |

## 4.2 ขั้นตอนที่ 2: หาบริษัท → แกะงบ

เมื่อรู้ว่ากลุ่มไหนได้ประโยชน์ → หาบริษัทที่ดีที่สุดในกลุ่มนั้น

อย่าดูแค่ชื่อหรือแบรนด์ ดูตัวเลขงบการเงิน เพราะงบไม่โกหก ผู้บริหารอาจโกหกได้ แต่งบที่ได้รับการตรวจสอบโกหกยากกว่า

**Quick Screen 5 ข้อ:**

| ข้อ | ดู | ดี | ระวัง |
|-----|-----|-----|-------|
| 1 | กำไรสุทธิ 4Q ล่าสุด | เติบโตทุก Q | ลดลง หรือสลับกำไร-ขาดทุน |
| 2 | P/E | ต่ำกว่าค่าเฉลี่ยกลุ่ม | สูงกว่ากลุ่ม > 50% |
| 3 | P/BV | < 1.5 | > 3 |
| 4 | หนี้สิน/ทุน (D/E) | < 1 | > 2 |
| 5 | ปันผล | จ่ายสม่ำเสมอ ≥ 3 ปี | ไม่เคยจ่าย หรือหยุดจ่าย |

ผ่าน 4/5 ข้อ = น่าสนใจ → ให้รหัส SQ หน้า A
ผ่าน 2-3 ข้อ = ปานกลาง → รหัส B
ผ่าน 0-1 ข้อ = หลีกเลี่ยง → รหัส C

**แกะงบเพิ่มเติม — ถ้ามีเวลา:**

```
┌───────────────────────────────────────────────────────┐
│  Gross Margin และ Net Margin                           │
│                                                       │
│  Gross Margin = (รายได้ - ต้นทุนขาย) / รายได้         │
│  ดีขึ้น = บริษัทควบคุมต้นทุนได้ดีขึ้น                  │
│  แย่ลง = ต้นทุนสูงขึ้น หรือราคาขายลด                  │
│                                                       │
│  Net Margin = กำไรสุทธิ / รายได้                       │
│  ดี = กำไรสูงต่อยอดขาย                                 │
│  แย่ = ค่าใช้จ่ายบริหารสูง หรือดอกเบี้ยกินมาก         │
└───────────────────────────────────────────────────────┘
```

```
┌───────────────────────────────────────────────────────┐
│  Cash Flow — สิ่งที่โกหกยากที่สุด                     │
│                                                       │
│  Operating Cash Flow (OCF) > กำไรสุทธิ               │
│  = กำไรที่ได้มาเป็นเงินสดจริง ✓                       │
│                                                       │
│  OCF < กำไรสุทธิ มาก ๆ                                │
│  = กำไรบนกระดาษ แต่เงินสดไม่มา = ระวัง               │
│                                                       │

## 4.3 ขั้นตอนที่ 3: อ่านกราฟ + เทียบดัชนี

**อ่านกราฟ — ให้รหัสหลัง:**
- ดูเทรนด์ (EMA 35 เป็นเส้นหลัก)
- ดูแนวรับ-แนวต้าน (จุดที่ราคาเด้งซ้ำ)
- ดู Volume (เพิ่มเมื่อขึ้น = healthy / เพิ่มเมื่อลง = ขายหนัก)
- ให้รหัส SQ หลัง (A/B/C)

**เทียบดัชนี SET กับหุ้น — สำคัญมาก:**

```
┌──────────────────────────────────────────────────────────┐
│              SET INDEX vs หุ้นรายตัว                      │
├─────────────────────┬────────────────────────────────────┤
│ SET Break ขึ้น      │ หุ้นใหญ่ที่จ่อ Break → Break จริง  │
│                     │ หุ้นเล็กวิ่งแรงกว่า SET            │
│                     │ → เข้าได้ (ถ้า SQ ดี)              │
│                     │ → เพิ่มขนาดได้                     │
├─────────────────────┼────────────────────────────────────┤
│ SET พักตัว          │ หุ้นใหญ่พัก                        │
│                     │ หุ้นเล็กยังเล่นได้ (rotation)      │
│                     │ → เลือกหุ้นเล็กที่ SQ ดี           │
│                     │ → ลดขนาดหุ้นใหญ่                   │
├─────────────────────┼────────────────────────────────────┤
│ SET ลง              │ หุ้นใหญ่ถ้ายืนได้ → รอ SET กลับ   │
│                     │ หุ้นเล็กอาจยังเล่นได้ แต่ไม่แรง   │
│                     │ → ลดขนาดทุกตัว                     │
│                     │ → เพิ่มเงินสด                      │
└─────────────────────┴────────────────────────────────────┘
```

**TIME ZONE ตลาดหุ้นไทย — รู้จักจังหวะ:**

```
┌───────────────────────────────────────────────────────┐

│                                                       │

│  ↓  ช่วงเงียบ Volume ต่ำ อย่าตัดสินใจ                 │

| TIME ZONE | ดูอะไร | บอกอะไร |
|-----------|--------|--------|

## 4.4 ขั้นตอนที่ 4: แกะ Bid-Offer

ก่อนกดซื้อ ดู Bid-Offer เป็นด่านสุดท้าย ตาราง Bid-Offer บอกให้รู้ว่า "ตอนนี้แรงซื้อ-ขายเป็นอย่างไร"

**4 สถานะ Bid-Offer:**

| สถานะ | อ่านว่า | ทำอะไร |
|-------|--------|--------|
| Bid บาง Offer บาง | ต้นเทรนด์ ยังเงียบ ไม่มีคนสนใจ | ถ้ามี Ticker Buy เข้าต่อเนื่อง → น่าสนใจ |
| Bid บาง Offer หนา | MM อาจกำลังเก็บของ | ถ้าผ่านแนวต้านได้ → ขึ้นแรง / ถ้าไม่ผ่าน → รอ |
| Bid หนา Offer บาง | ปลายเทรนด์ MM ปล่อยของ | ระวัง ดูเหมือนแข็ง แต่อาจจะพัง |
| Bid หนา Offer หนา | Volume Peak | ต้นเทรนด์ = Breakout / ปลายเทรนด์ = ปล่อยของ |

**วิธีอ่าน Bid-Offer เร็ว:**

```
┌───────────────────────────────────────────────────────┐
│  Bid-Offer Cheat Sheet                                │
│                                                       │
│  ขั้นที่ 1: Bid บาง หรือหนา?                         │
│  ขั้นที่ 2: Offer บาง หรือหนา?                       │
│  ขั้นที่ 3: Ticker — Buy หรือ Sell ที่กำลังเดิน?      │
│  ขั้นที่ 4: ราคาอยู่ที่ไหน — ต้นเทรนด์/กลาง/ปลาย?   │
│                                                       │
│  ผสมข้อมูลทั้ง 4 ถึงจะตอบได้ว่า "เข้าหรือรอ"        │
└───────────────────────────────────────────────────────┘
```

**ดู Ticker ประกอบ:**
- Ticker Buy ต่อเนื่อง ทีละ 500-2,000 หุ้น = มีคนเก็บเงียบ
- Ticker Sell ก้อนใหญ่ ๆ ทีเดียว = มีคนทิ้ง
- Ticker Buy ก้อนใหญ่ช่วงปลายวัน = MM เก็บก่อนปิด
- Ticker สลับ Buy-Sell เท่า ๆ กัน = washing ไม่มีทิศ

**Volume ขั้นต่ำก่อนพิจารณา:**
- มูลค่าซื้อขาย ≥ 50 ล้านบาท/วัน
- หรือจำนวนหุ้น ≥ 1 ล้านหุ้น/วัน
- ต่ำกว่านี้ = สภาพคล่องไม่พอ ซื้อง่ายแต่ขายยาก

**ตัวอย่าง — อ่าน Bid-Offer ก่อนกดซื้อ:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: Bid-Offer ที่หลอกตา                         │
├──────────────────────────────────────────────────────────┤
│  หุ้น XYZ ราคา 10.00 บาท อยู่ในขาขึ้น                   │
│                                                          │
│  คุณเห็น:                                                │
│  Bid: 500,000 หุ้น ที่ 9.90 (หนามาก!)                    │
│  Offer: 20,000 หุ้น ที่ 10.10 (บางมาก!)                  │
│                                                          │
│  คนทั่วไปคิด:                                            │
│  "Bid หนา Offer บาง = มีคนรับเยอะ = หุ้นจะขึ้น"         │
│  → ซื้อ                                                  │
│                                                          │
│  ผู้รอดคิด:                                               │
│  "Bid หนา Offer บาง = ปลายเทรนด์? MM ปล่อยของ?"        │
│  → ดู Ticker: Buy เข้ามาทีละ 2,000-5,000 หุ้น (รายย่อย) │
│  → ดู Volume: สูงผิดปกติ (3x ค่าเฉลี่ย)                 │
│  → ดูราคา: อยู่สูงมา 3 เดือน ไม่ใช่ต้นเทรนด์             │
│                                                          │
│  สรุป: Bid หนานั้นอาจเป็น "Bid ดัก"                      │
│  MM วาง Bid เพื่อให้คนรู้สึกมั่นใจ                       │
│  แล้ว MM ค่อย ๆ ขายผ่าน Offer ทีละนิด                    │
│  เมื่อขายหมด → Bid หนาจะหายไป → ราคาร่วง               │
│                                                          │
│  ทำ: ไม่ซื้อ รอดูอีก 1-2 วัน                             │
│  ผลลัพธ์: 2 วันต่อมาราคาลง 8% Bid หนาหายไปจริง          │
└──────────────────────────────────────────────────────────┘
```

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: Bid บาง Offer หนา — มือ MM กำลังเก็บ       │
├──────────────────────────────────────────────────────────┤
│  หุ้น ABC ราคา 5.00 บาท ไซด์เวย์มา 2 เดือน              │
│  SQ = AB (พื้นฐาน A ยังไม่ Break)                       │
│                                                          │
│  Bid: 50,000 หุ้น ที่ 4.90 (บาง)                         │
│  Offer: 800,000 หุ้น ที่ 5.10 (หนามาก!)                  │
│                                                          │
│  Ticker: Buy เข้าต่อเนื่อง ทีละ 1,000-3,000 หุ้น         │
│  Volume: 80 ล้านบาท/วัน (ค่าเฉลี่ย 30 ล้าน → 2.7x)     │
│                                                          │
│  วิเคราะห์:                                              │
│  Offer หนาที่ 5.10 = "กำแพง" ที่รอให้รายย่อยซื้อเติม     │
│  Ticker Buy เงียบ ๆ = มีคนสะสมอยู่                       │
│  Volume เพิ่ม 2.7x = ความสนใจเพิ่มขึ้น                   │
│                                                          │
│  ทำ: ซื้อ 15% ของพอร์ต ต้องการดู Break ที่ 5.10          │
│  ถ้า Offer ที่ 5.10 หมด → Break → เพิ่มอีก 10%          │
│  ถ้าไม่ Break ใน 1 สัปดาห์ → ออก                        │
│                                                          │
│  ผลลัพธ์: 3 วันต่อมา Break 5.10 ราคาวิ่งไป 6.20 (+24%)  │
└──────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────┐
│  KEY CONCEPT                            │
│                                         │
│  Bid-Offer บอก "ตอนนี้"                 │
│  SQ บอก "พื้นฐาน + เทรนด์"             │
│                                         │
│  ต้องใช้ทั้งสอง                          │
│  SQ ดีแต่ Bid-Offer เตือน → รอก่อน     │
│  Bid-Offer ดีแต่ SQ แย่ → ไม่เข้า      │
└─────────────────────────────────────────┘
```

## 5.1 Stop Hunt (ไม้ตกใจ)

**คืออะไร:** หุ้นถูก sweep Stop Loss ลงแรง → แล้วกลับขึ้นทันที คนที่ตั้ง Stop ไว้ต่ำโดนกินออก แล้วหุ้นวิ่งต่อโดยไม่มีเขา

**เงื่อนไข — ต้องครบทุกข้อ:**
1. หุ้นอยู่ในเทรนด์ขาขึ้น (SQ = xA) — ถ้าไม่ใช่ขาขึ้น อย่าเข้า
2. ราคาลงแรงกะทันหัน ภายใน 5-15 นาที — ไม่ใช่ลงค่อย ๆ ทั้งวัน
3. Volume ตอนลง "ไม่สูงผิดปกติ" — ถ้า Volume สูง = ขายจริง ไม่ใช่ sweep
4. ลงมาแตะแนวรับสำคัญ (EMA 15 / แนวรับก่อนหน้า) แล้วเด้งทันที
5. Bid กลับมาหนา หลังจาก Sell หยุด — สัญญาณว่ามีคนรับ

**จังหวะเข้า:**
- รอจนราคาเด้งกลับเหนือแนวรับ ไม่ใช่ซื้อตอนกำลังลง
- Bid เริ่มหนา + candle กลับเป็นเขียว → เข้า

**Exit:**
- Target: กลับไปที่ราคาก่อน sweep (100% recovery)
- Stop Loss: ต่ำกว่าจุดต่ำสุดที่ sweep ลงไป 1-2%

**Risk:Reward:** ปกติ 1:2 ถึง 1:3

## 5.2 Breakout

**คืออะไร:** หุ้นทะลุแนวต้านสำคัญที่กดมานาน → เริ่มเทรนด์ใหม่

**เงื่อนไข — ต้องครบทุกข้อ:**
1. หุ้นอยู่ในกรอบ (sideway) มาอย่างน้อย 2-4 สัปดาห์
2. มีแนวต้านชัดเจน — ราคาเด้งลงจากจุดเดิม ≥ 3 ครั้ง (2 ครั้งพอได้ แต่ 3 ดีกว่า)
3. Volume เพิ่มขึ้นอย่างชัดเจนในวันที่ทะลุ ≥ 2x ค่าเฉลี่ย — ถ้าน้อยกว่านี้ระวัง Fake
4. ราคาปิดเหนือแนวต้าน ไม่ใช่แค่ทิ่มแล้วกลับ
5. Bid-Offer: Bid หนา Offer หนา ตอน break = Volume Peak ต้นเทรนด์

**จังหวะเข้า:**
- **แบบ A (aggressive):** ซื้อเมื่อทะลุ + Volume confirm ในวันเดียวกัน — ได้ราคาดีกว่า แต่เสี่ยง Fake Break
- **แบบ B (conservative):** รอ pullback กลับมาทดสอบแนวต้านเดิม (ที่กลายเป็นแนวรับ) แล้วซื้อ — ปลอดภัยกว่า

**Exit:**
- Target: ใช้ Fibonacci Extension (127.2% หรือ 161.8% ของกรอบ sideway)
- Stop Loss: ต่ำกว่าแนวต้านที่เพิ่ง break (ถ้ากลับลงมาใต้ = break ปลอม)

**Risk:Reward:** ปกติ 1:2 ถึง 1:4

│  1. Volume ไม่สูงพอ (ต้อง ≥ 2x)                         │
│  2. ราคาปิดใต้แนวต้าน (ต้องปิดเหนือ)                    │
│  3. ทะลุแค่นิดเดียว (0.10 บาท = 1%)                     │
│                                                          │
│  ถ้าซื้อตอน 9.10 → วันรุ่งขึ้นราคาลงไป 8.50 = -6.6%    │
│  ถ้ารอ pullback → ไม่ได้ซื้อ เพราะราคาไม่กลับมาเหนือ 9  │
│  → การรอช่วยหลีกเลี่ยง false break                      │
└──────────────────────────────────────────────────────────┘
```

**Checklist ก่อนเข้า Breakout:**

☐ หุ้น sideway มาอย่างน้อย 2 สัปดาห์
☐ แนวต้านชัด เด้งลงกลับ ≥ 2 ครั้ง
☐ Volume ≥ 2x ค่าเฉลี่ยในวันที่ break
☐ ราคาปิดเหนือแนวต้าน
☐ ถ้า aggressive: ซื้อพร้อม Volume confirm
☐ ถ้า conservative: รอ pullback มาทดสอบแนวรับใหม่แล้วเด้ง

## 5.3 Buy on Dip

**คืออะไร:** หุ้นที่อยู่ในเทรนด์ขาขึ้น ย่อตัวลงมาที่แนวรับ → ซื้อตอนย่อ รอขึ้นต่อ

**ปลอดภัยที่สุดสำหรับมือใหม่** — Stop แคบ Reward ไกล เทรนด์หนุน

**เงื่อนไข — ต้องครบทุกข้อ:**
1. หุ้นอยู่ในเทรนด์ขาขึ้น (Higher High + Higher Low) ชัดเจน
2. ราคาย่อลงมาแตะแนวรับ: EMA 15 / EMA 35 / Fibonacci 38.2%-61.8% / Trendline
3. Volume ลดลงตอนย่อ — ไม่ใช่ขายจริง แค่พักตัว
4. มีสัญญาณกลับตัว: แท่งเทียน Hammer / Doji / Bullish Engulfing + Volume เพิ่มวันกลับ

**จังหวะเข้า:**
- ซื้อเมื่อราคาแตะแนวรับ + มีสัญญาณกลับตัว
- ไม่ซื้อตอนกำลังลง — รอดูก่อนว่า "หยุดลง" จริง

**Exit:**
- Target: High เดิม หรือ Higher High ใหม่ หรือ Fibonacci Extension 161.8%
- Stop Loss: ต่ำกว่าแนวรับที่ซื้อ 2-3% (ถ้าหลุดแนวรับ = เทรนด์อาจเปลี่ยน)

**Risk:Reward:** ปกติ 1:3 ถึง 1:5 (เพราะ stop แคบ target ไกล)

## 5.4 Channel Trade

**คืออะไร:** หุ้นแกว่งในกรอบ (sideway) → ซื้อที่แนวรับล่าง ขายที่แนวต้านบน ซ้ำไปเรื่อย ๆ จนกว่ากรอบจะสิ้นสุด

**เงื่อนไข — ต้องครบทุกข้อ:**
1. หุ้นอยู่ในกรอบชัดเจน ≥ 2 สัปดาห์ (SQ = xB)
2. แนวรับ-แนวต้านชัด ราคาเด้งจากจุดเดิม ≥ 3 ครั้งทั้งบนและล่าง
3. กรอบกว้างพอทำกำไร ≥ 5-7% ระหว่างแนวรับ-แนวต้าน
4. Volume ต่ำตลอด ไม่มีแรงซื้อขายผิดปกติ

**จังหวะเข้า:**
- ซื้อที่แนวรับ (ขอบล่างของกรอบ) + รอสัญญาณกลับตัว
- ขายที่แนวต้าน (ขอบบน) + อาจถอยออกบางส่วน

**Exit:**
- Target: แนวต้าน (ขอบบน)
- Stop Loss: ต่ำกว่าแนวรับ 1-2% (ถ้าหลุดกรอบลง = ไม่ใช่ sideway อีกต่อไป)

**Risk:Reward:** ปกติ 1:2 ถึง 1:3

## 6.1 ประเภท 1: ความเสี่ยงต่ำ + ผลตอบแทนมั่นคง

**น้ำหนักมากที่สุด** — นี่คือฐานที่ต้องสร้างก่อนทุกอย่าง

เป้าหมายของประเภท 1: สร้าง passive income ที่ "ครอบค่าใช้จ่ายพื้นฐาน" ได้

เมื่อ passive income ≥ ค่าใช้จ่ายรายเดือน = ไม่ต้องกังวลแม้ประเภท 3 ขาดทุน

| สินทรัพย์ | ผลตอบแทน/ปี | ความเสี่ยง | หมายเหตุ |
|-----------|-------------|-----------|----------|
| พันธบัตรรัฐบาล | 2-4% | ต่ำมาก | เงินต้นไม่หาย รัฐค้ำ |
| หุ้นกู้ Investment Grade A | 3-5% | ต่ำ | ดูเครดิตเรตติ้งก่อนซื้อ |
| REITs | 5-8% | ต่ำ-ปานกลาง | ปันผลสม่ำเสมอ กระจายในหลายโครงการ |
| หุ้นปันผลมั่นคง | 4-7% | ปานกลาง | เลือกที่จ่ายต่อเนื่อง ≥ 5 ปี |
| อสังหาฯ ให้เช่า | 5-8% | ปานกลาง | passive income จริง แต่สภาพคล่องต่ำ |
| ประกันสะสมทรัพย์ | 2-3% | ต่ำมาก | สภาพคล่องต่ำมาก แต่บังคับออม |

**สัดส่วนตามอายุ:**
- อายุ < 30: 20-30% ของ net worth — ยังมีเวลาเสี่ยง
- อายุ 30-40: 30-50% — เริ่มสร้างฐาน
- อายุ 40+: 50-70% — ฐานต้องแข็งแกร่งก่อน

```
┌─────────────────────────────────────────┐
│  KEY CONCEPT                            │
│                                         │
│  ประเภท 1 คือ "เงินที่ทำงานแทนเรา"     │
│  ไม่ต้องนั่งจ้องจอ ไม่ต้องตัดสินใจ     │
│                                         │
│  เป้าหมาย:                             │
│  Passive Income ≥ ค่าใช้จ่าย/เดือน    │
│  = ไม่ต้องกลัวขาดทุนจากประเภท 3       │
└─────────────────────────────────────────┘
```

## 6.2 ประเภท 2: Asset ใช้ประโยชน์

**น้ำหนักน้อยที่สุด** — มีเท่าจำเป็น อย่าเกิน

| สินทรัพย์ | ใช้ประโยชน์ | ปัญหา | กฎ |
|-----------|-----------|------|-----|
| บ้าน/คอนโดอยู่เอง | ที่อยู่อาศัย | ไม่สร้างรายได้ ผ่อนกินรายเดือน | ผ่อนไม่เกิน 30% ของรายได้ |

```
┌─────────────────────────────────────────┐

│                                         │

└─────────────────────────────────────────┘
```

**ข้อยกเว้น — เมื่อประเภท 2 กลายเป็นประเภท 1:**

- บ้าน/คอนโดที่ปล่อยเช่า → ให้ Yield → ย้ายไปนับเป็นประเภท 1
- เงื่อนไข: ค่าเช่า > ค่าผ่อน + ค่าดูแล = yield เป็นบวก

## 6.3 ประเภท 3: Capital Gain (เงินทำงาน)

## 6.5 Position Sizing + Sniper Shot + Hit Rate + RRR

**Sniper Shot — ไม่เกิน 3 ตัวพร้อมกัน:**

| จำนวนหุ้น | ปัญหา |
|-----------|--------|
| 1 ตัว | เสี่ยง concentrated — ถ้าผิด เจ็บหนัก |
| 2-3 ตัว | พอดี ดูทันจัดการได้ — เป้าหมาย |
| 5+ ตัว | ดูไม่ทัน ขายไม่ทันเวลาลง |
| 10+ ตัว | เหมือนซื้อกองทุน แต่ไม่มีผู้จัดการ |

**Position Sizing ตาม SQ:**

| SQ | ขนาดไม้ |
|----|---------|
| AA | Full size (ตาม 2% rule) |
| AB / BA | Half size |
| BB / AC | Quarter size |
| BC / CB / CC | ไม่เข้า |

```
┌─────────────────────────────────────────┐
│  KEY CONCEPT                            │
│                                         │
│  ยิ่ง SQ ดี → ยิ่งใส่มาก              │
│  ยิ่ง SQ แย่ → ยิ่งใส่น้อย หรือไม่เข้า │
│                                         │
│  ไม่ใช่เพราะ "อยากกำไรมาก"             │
│  แต่เพราะ "probability สูงกว่า"        │
└─────────────────────────────────────────┘
```

**Hit Rate — ชนะกี่ไม้จาก 10:**

| Hit Rate | สถานะ | ทำอะไร |
|----------|--------|--------|
| < 50% | ไม่ควรเทรดเงินจริง | กลับไป Paper Trade |
| 50-60% | พอได้ แต่ต้อง RRR ≥ 1:3 | เงินจริงขนาดเล็ก |
| 60-70% | ดี เริ่มมั่นใจได้ | เพิ่มขนาดได้ |
| > 70% | พร้อมเป็น Full Time Trader | ขยายพอร์ตได้ |

**RRR (Risk:Reward Ratio) — ต้องไม่ต่ำกว่า 1:3:**

```
  ถ้า Stop Loss = 3% (เสียไม่เกิน 3%)
  Target ต้อง ≥ 9% (ได้อย่างน้อย 9%)
  → RRR = 1:3

  แปลว่า: แม้ผิด 7 ครั้งจาก 10
  ถ้าถูก 3 ครั้ง ก็ยังกำไร

  ผิด 7 ครั้ง × 3% = -21%
  ถูก 3 ครั้ง × 9% = +27%
  สุทธิ = +6%
```

> ⚠️ WARNING: Hit Rate สูงไม่มีความหมายถ้า RRR ต่ำ ชนะ 9/10 แต่แพ้ทีเดียวหนัก = ขาดทุน สองตัวนี้ต้องดูคู่กันเสมอ

**Position Sizing — ใส่ไม้ละเท่าไหร่:**

กฎ: ห้ามเสียเกิน 2% ของพอร์ตต่อ 1 ไม้

```
  พอร์ต = 1,000,000 บาท
  Max loss ต่อไม้ = 2% = 20,000 บาท

  ถ้า Stop Loss = 5%
  → ขนาดไม้ = 20,000 ÷ 5% = 400,000 บาท (40% ของพอร์ต)

  ถ้า Stop Loss = 10%
  → ขนาดไม้ = 20,000 ÷ 10% = 200,000 บาท (20% ของพอร์ต)

  ถ้า Stop Loss = 3%
  → ขนาดไม้ = 20,000 ÷ 3% = 666,667 บาท (67% ของพอร์ต)
  → แต่! Sniper Shot บอกว่าไม่เกิน 50% ต่อตัว
  → ดังนั้นใส่ 500,000 max
```

**ตัวอย่าง — คำนวณ Position Size จริงก่อนซื้อ:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: คำนวณ Position Size ก่อนซื้อ DELTA          │
├──────────────────────────────────────────────────────────┤
│  พอร์ต: 2,000,000 บาท                                    │
│  Max loss ต่อไม้: 2% = 40,000 บาท                        │
│                                                          │
│  DELTA SQ = AA → Full size                               │
│  ราคาซื้อ: 850 บาท                                       │
│  Stop Loss: 820 บาท (ใต้ EMA 35 + Higher Low ก่อนหน้า)  │

## 7.2 Leading Diagonal vs Ending Diagonal

Diagonal = คลื่นที่ "ดูเหมือน impulse แต่โครงสร้างข้างในต่าง"

**Leading Diagonal:**

เกิดที่คลื่น 1 หรือคลื่น A = จุดเริ่มต้นของ move ใหญ่

โครงสร้างภายใน: 5-3-5-3-5 (motive) หรือ 3-3-3-3-3 (corrective zigzag)

```
     ╱╲
    ╱  ╲
   ╱    ╲     ╱  ← คลื่น 5 ของ diagonal
  ╱   ╱╲ ╲   ╱
 ╱   ╱  ╲ ╲ ╱
╱   ╱    ╲ V  ← คลื่น 4
   ╱      ╲
  ╱        ╲

  ลักษณะ: เส้นบน-ล่างบรรจบกัน (contracting)
  หรือ: เส้นบน-ล่างแยกออก (expanding)
```

| ประเภท | ลักษณะ | ความหมาย | ความเสี่ยง |
|--------|--------|---------|-----------|
| Contracting | คลื่น 1 ยาวที่สุด กรอบแคบลงเรื่อย ๆ | เริ่มต้นเทรนด์ใหม่ แบบค่อยเป็นค่อยไป | ปานกลาง |
| Expanding | คลื่นสุดท้ายยาวที่สุด กรอบกว้างขึ้น | เริ่มต้นรุนแรง ผันผวนสูง | สูง |

**Ending Diagonal:**

เกิดที่คลื่น 5 หรือคลื่น C = จุดสิ้นสุดของ move ใหญ่

โครงสร้างภายใน: 3-3-3-3-3 (ทุกคลื่นย่อยเป็น zigzag)

สำคัญมาก: เมื่อเห็น Ending Diagonal = ใกล้จบเทรนด์ → เตรียมกลับทิศ → Sharp Reversal มักตามมา

```
┌─────────────────────────────────────────┐
│  KEY CONCEPT                            │
│                                         │
│  Leading Diagonal ที่คลื่น 1            │
│  = เริ่มต้นเทรนด์ใหม่ → เตรียมเข้า    │
│                                         │
│  Ending Diagonal ที่คลื่น 5             │
│  = จบเทรนด์ → เตรียมออก                │
│  → Sharp Reversal ตามมา                 │
│  → คนที่ซื้อตอน Ending Diagonal         │
│    มักซื้อที่จุดสูงสุดของเทรนด์         │
└─────────────────────────────────────────┘
```

**สัญญาณยืนยัน Ending Diagonal:**

```

```

## 7.4 Fibonacci Retracement + Extension

**Fibonacci Retracement — หาจุดพักตัว:**

ใช้เมื่อราคาย่อตัว (pullback) → หาว่าจะย่อถึงไหน

| Fibo Level | ความหมาย | Elliott Wave |
|------------|---------|------------|
| 23.6% | ย่อตื้นมาก (เทรนด์แรงมาก) | คลื่น 4 ที่แรง |
| 38.2% | ย่อปกติ | คลื่น 4 ทั่วไป |
| 50.0% | ย่อกลาง | จุดกึ่งกลาง |
| 61.8% | ย่อลึก | คลื่น 2 ทั่วไป |
| 78.6% | ย่อลึกมาก (ใกล้ล้มเลิก) | คลื่น 2 ที่รุนแรง |

**วิธีลาก Fibonacci Retracement:**

```
  1. หาจุด Low (จุดเริ่มต้นขาขึ้น)
  2. หาจุด High (จุดสูงสุดก่อนย่อ)
  3. ลาก Fibo จาก Low → High
  4. ดูว่าราคาย่อมาถึง level ไหน
  5. ซื้อเมื่อ:
     - ราคาแตะ Fibo level
     - มีสัญญาณกลับตัว (Hammer / Doji)
     - Volume เพิ่มขึ้น
     - Bid เริ่มหนา
```

```
┌─────────────────────────────────────────┐
│  KEY CONCEPT                            │
│                                         │
│  Fibo ไม่ใช่เส้นวิเศษ                  │
│  ใช้เป็น "โซน" ไม่ใช่ "จุด"             │
│                                         │
│  ซื้อเมื่อ Fibo level + สัญญาณอื่น     │
│  ยืนยันพร้อมกัน                         │
│  Fibo อย่างเดียว = ไม่พอ               │
└─────────────────────────────────────────┘
```

**Fibonacci Extension — หาเป้าราคา:**

ใช้เมื่อราคา break ขึ้นต่อ → หาว่าจะขึ้นถึงไหน

| Fibo Extension | ใช้เป็น | Elliott Wave |
|----------------|--------|------------|
| 100% | เป้าแรก (ปลอดภัย) | คลื่น 5 สั้น |
| 127.2% | เป้ากลาง | คลื่น 3 ทั่วไป |
| 161.8% | เป้าไกล (Golden Ratio) | คลื่น 3 ปกติ |
| 200% | เป้าไกลมาก | คลื่น 3 ที่แรงมาก |
| 261.8% | Extended wave | คลื่น 3 ที่แรงมาก ๆ |

```
┌─────────────────────────────────────────┐
│  ACTION                                 │
│                                         │
│  ใช้ Fibo 2 จังหวะ:                     │
│                                         │
│  1. ราคาย่อ → ลาก Retracement          │
│     → ซื้อที่โซน 38.2% หรือ 61.8%      │
│                                         │
│  2. ราคา break ขึ้น → ลาก Extension    │
│     → ตั้งเป้าที่ 127.2% หรือ 161.8%   │
│                                         │
│  Stop Loss:                             │
│  ซื้อที่ 38.2% → SL ใต้ 50.0%          │
│  ซื้อที่ 61.8% → SL ใต้ 78.6%          │
└─────────────────────────────────────────┘
```

> ⚠️ WARNING: Fibonacci ไม่ใช่เส้นวิเศษ ราคาไม่ได้หยุดตรง Fibo level ทุกครั้ง ใช้เป็น "โซน" ไม่ใช่ "จุด" — ถ้าราคาอยู่ใกล้ Fibo level + มีสัญญาณอื่นยืนยัน (แท่งเทียน, Volume, Bid-Offer) → ถึงจะเข้า

│  คน A ขาดทุน 15% + พลาดเด้ง = ขาดทุนจริง ~25%          │
│  คน B ขาดทุน 2% = แทบไม่เสียอะไร                        │
│                                                          │
│  ต่างกันตรงไหน?                                          │
│  - ลดขนาดมาก่อน (ข้อ 4: จำลอง worst case)               │
│  - มี Stop Loss ทุกตัว (ข้อ 3)                           │
│  - ไม่ตกใจ ไม่ขายหมดตอนต่ำสุด (ข้อ 8)                  │
│  - ปิดจอ ไม่ตัดสินใจตอนใจร้อน                            │
└──────────────────────────────────────────────────────────┘
```

## 8.2 Circuit Breaker 3 ระดับ

ข้อตกลงกับตัวเอง ทำตอนใจเย็น ใช้ตอนใจร้อน

```

│              CIRCUIT BREAKER ส่วนตัว                     │
├──────────┬──────────────────┬────────────────────────────┤
│ Level    │ เงื่อนไข         │ Action                    │
├──────────┼──────────────────┼────────────────────────────┤
│ Level 1  │ พอร์ตลง 3%      │ หยุดซื้อขาย 1 วัน         │
│  ⚡      │ ภายใน 1 วัน      │ ปิดจอ ทบทวนทุกไม้          │
│          │                  │ เขียนลงกระดาษว่าผิดอะไร    │
├──────────┼──────────────────┼────────────────────────────┤
│ Level 2  │ พอร์ตลง 10%     │ คัทหมด                    │
│  🔴      │ สะสม             │ หยุด 1 สัปดาห์            │
│          │                  │ ทบทวนว่าทำอะไรผิด          │
│          │                  │ กลับมาด้วยขนาดเล็กครึ่งหนึ่ง│
├──────────┼──────────────────┼────────────────────────────┤

│          │                  │ กลับมาด้วย Paper Trade ก่อน│
│          │                  │ ฝึกจนนิ่ง ≥ 3 เดือน        │

## 8.3 Stop Loss 4 วิธี

เลือก 1-2 วิธีที่เหมาะกับสไตล์ ใช้ให้ชำนาญ

**วิธีที่ 1 — Dow Theory:**

ขาขึ้น = Higher High + Higher Low

Stop Loss = ต่ำกว่า Higher Low ล่าสุด

```
  ราคา
   │      ╱╲  HH
   │     ╱  ╲     ╱
   │  ╱╲╱    ╲   ╱
   │ ╱  HL    ╲ ╱ HL
   │╱          V
   │         ↑ Stop Loss = ใต้ HL นี้
   └──────────────── เวลา
```

ข้อดี: สอดคล้องกับนิยาม "ขาขึ้น" — ถ้า Stop ถูก trigger = นิยามเปลี่ยน → ออกถูก
ข้อเสีย: ถ้า HL ห่างมาก Stop ก็ไกล → ใส่ขนาดได้น้อยลง

**วิธีที่ 2 — Trendline:**

ลากเส้น trendline เชื่อม Low ที่สูงขึ้น

Stop Loss = เมื่อราคาปิดใต้ trendline

ข้อดี: ปรับตัวตามเทรนด์ ยิ่งขึ้นนาน trendline ยิ่งสูง → stop ยิ่งสูง (ล็อกกำไรได้)
ข้อเสีย: ต้องลาก trendline ให้ถูก ถ้าลากผิด stop ผิดตาม

**วิธีที่ 3 — EMA:**

Stop Loss = เมื่อราคาปิดต่ำกว่า EMA ที่เลือก

| EMA | เหมาะกับ | ตอบสนอง |
|-----|---------|--------|
| EMA 5 | Daytrade (1-3 วัน) | เร็วมาก สั่น |
| EMA 15 | Swing Trade (1-2 สัปดาห์) | ปานกลาง |
| EMA 35 | Position Trade (1-2 เดือน) | ช้า แต่กรองสัญญาณปลอม |

ข้อดี: ง่าย ดูได้ทันที ไม่ต้องวิเคราะห์มาก
ข้อเสีย: สาย สั่น (whipsaw) ในตลาด sideway

**วิธีที่ 4 — % ที่รับได้:**

กำหนดล่วงหน้า: "ผมยอมเสียได้ ___% ต่อไม้"

| Risk Tolerance | Stop Loss | ใช้กับ |
|---------------|-----------|-------|
| เสี่ยงต่ำ | 3% | มือใหม่ / พอร์ตเล็ก |
| ปานกลาง | 5% | ส่วนใหญ่ |
| เสี่ยงสูง | 7-10% | หุ้น volatile / Position Trade |

ข้อดี: ง่ายที่สุด คำนวณขนาดได้ทันที
ข้อเสีย: ไม่สนใจแนวรับ อาจโดนกินก่อนเวลาถ้าวางไม่ถูก

```
┌─────────────────────────────────────────┐
│  ACTION                                 │
│                                         │
│  เลือก 1 วิธี Stop Loss หลัก           │
│  + 1 วิธีสำรอง                          │
│                                         │
│  แนะนำ:                                 │
│  Daytrade → EMA 5 + % (5%)             │
│  Swing → Trendline + EMA 15            │
│  Position → Dow Theory + EMA 35        │
│                                         │
│  ก่อนซื้อทุกครั้ง เขียน:                │
│  "Stop Loss ของไม้นี้ = ราคา ____"      │
│                                         │
│  ถ้าเขียนไม่ได้ = ยังไม่พร้อมซื้อ       │
└─────────────────────────────────────────┘
```

**ตัวอย่าง — Stop Loss 4 วิธีในหุ้นเดียวกัน:**

```
┌──────────────────────────────────────────────────────────┐
│  CASE STUDY: ซื้อหุ้นกลุ่มพลังงาน 50 บาท               │

├──────────────────────────────────────────────────────────┤
│  ราคาซื้อ: 50 บาท                                        │
│  Higher Low ล่าสุด: 47 บาท                               │
│  EMA 35: 46 บาท                                          │
│  Trendline: 48 บาท                                       │
│                                                          │
│  วิธีที่ 1 (Dow Theory): SL = 47 = -6%                   │
│  วิธีที่ 2 (Trendline):  SL = 48 = -4%                   │
│  วิธีที่ 3 (EMA 35):     SL = 46 = -8%                   │

│  สำคัญ: เลือก 1 วิธีแล้วใช้สม่ำเสมอ                     │
│  ห้ามเปลี่ยนวิธีหลังซื้อแล้ว เพราะจะย้ายเสา             │
└──────────────────────────────────────────────────────────┘
```

```

│    │ ไม่ซื้อเพราะ "ข้อมูลลับ" / "คนในบอก"               │

## Daytrade 4 รูปแบบ — เงื่อนไข 1 บรรทัด

```
  Stop Hunt  → ขาขึ้น + ลงแรง Volume ปกติ + เด้งกลับเหนือแนวรับ → เข้า
  Breakout   → Sideway ≥ 2 สัปดาห์ + Volume ≥ 2x ตอน Break → เข้า
  Buy on Dip → ขาขึ้น + ย่อถึง EMA/Fibo 38.2-61.8% + Volume ลด → เข้า
  Channel    → Sideway ชัด ≥ 2 สัปดาห์ + กรอบ ≥ 5% → ซื้อล่าง ขายบน
```

## Position Sizing — คำนวณขนาดก่อนกด Buy

```
  Max loss ต่อไม้  = 2% ของพอร์ต
  ขนาดไม้          = Max loss ÷ % Stop Loss
  ถือพร้อมกัน      ≤ 3 ตัว (Sniper Shot)

  ตัวอย่าง:
  พอร์ต 1,000,000 → Max loss = 20,000
  Stop = 5%       → ขนาดไม้ = 20,000 ÷ 5% = 400,000

  SQ Sizing:
  AA  = Full size  (ตาม 2% rule)
  AB/BA = Half     (ตาม 1% rule)
  BB/AC = Quarter  (ตาม 0.5% rule)
```

## Circuit Breaker — ทำทันทีเมื่อถึงระดับ

```
  -3%  ในวันเดียว  → หยุด 1 วัน ปิดจอ ทบทวนทุกไม้
  -10% สะสม        → คัทหมด หยุด 1 สัปดาห์ เขียนสิ่งที่ผิด
  -20% สะสม        → คัทหมด หยุด 1 เดือน กลับ Paper Trade ก่อน

  ไม่มีข้อยกเว้น ไม่มี "แต่ว่า..."
```

## Stop Loss 4 วิธี

```

  4. % ตายตัว    → 3% (ต่ำ) / 5% (กลาง) / 7-10% (สูง)

  เลือก 1 วิธีหลัก + 1 วิธีสำรอง ใช้ให้สม่ำเสมอ
```

## Survivor Mindset 10 ข้อ

```
  1.  ก่อนโต ต้องรอดก่อน            (ขาดทุน ≤ 3%/ไม้)
  2.  อย่าสู้สงครามที่แพ้แน่          (ถ้าทิศผิด ออก)
  3.  Stop Loss ทุกไม้                (ก่อนซื้อ เขียน SL ก่อน)
  4.  จำลอง worst case               (3 ไม้ผิดติดกัน = เสียเท่าไหร่?)
  5.  ขายไม้จิ้มฟัน อย่าขายสากกะเบือ (กำไรบ่อย สม่ำเสมอ > รอ big trade)
  6.  มึงมายังไง กูไปอย่างงั้น         (ปรับตามสถานการณ์)
  7.  อย่าสวนกระแส                   (ยกเว้น super fear/greed + RSI extreme)
  8.  ผิดแล้ว ล้างหน้าไพ่ใหม่ทันที    (ไม่เสียดาย ตัดออก เริ่มใหม่)
  9.  คิดถึงครอบครัว                  (เดือดร้อนไหม? → ลดขนาด / ไม่เข้า)
  10. อยู่รอดให้นาน เดี๋ยวมันโตเอง    (Compound ทำงานก็ต่อเมื่อยังอยู่)
```

## Bid-Offer 4 สถานะ

```
  Bid หนา / Offer บาง  → แรงซื้อมาก ราคามีแนวโน้มขึ้น
  Bid บาง / Offer หนา  → แรงขายมาก ราคามีแนวโน้มลง
  ทั้งคู่หนา           → Volume สูง มีแรงซื้อขายสมดุล
                         (ต้นเทรนด์ = ดี / ปลายเทรนด์ = ระวัง)
  ทั้งคู่บาง           → Volume ต่ำ Sideway ไม่มีทิศ
                         ไม่เหมาะเทรด รอ
```
